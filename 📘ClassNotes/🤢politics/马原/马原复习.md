#### 1. 马克思主义基本原理简介
马克思主义的基本组成： 马克思主义哲学，马克思主义政治经济学，科学社会主义

基本立场：以无产阶级的解放和全人类的解放为己任， 以认得自用全面发展为美好目标，以人民为中心，坚持一切为了人民，一切依靠人民，全心全意为人民谋幸福

 基本观点: p3

- 基本方法：建立在辩证唯物主义和历史唯物主义世界观和方法论基础上，知道正确认识世界和改造世界的思想方法， 主要包括：
	1.  实事求是的方法
	2.  辩证分析的方法
	3.  社会基本矛盾和主要矛盾
	4.  历史分析的方法
	5.  阶级分析的方法
	6.  群众路线的方法

创立的理论来源：三大思潮

1.  德国古典哲学
2.  英国古典政治经济学
3.  英法的空想社会主义

1848.2《共产党宣言》

鲜明特征：

**科学性，人民性，实践性，发展性**

1.  马克思主义以辩证唯物主义和历史唯物主义为基础，以事实为依据，规律为对象，实践为检验标准
2.  以无产阶级为基础，以解放全人类为目标
3.  理论与实践相结合

#### 2. 世界的物质性以及发展规律

- 对于意识和物质关系的分析
	- 唯物主义包括
		-  古代朴素唯物主义
		- 近代形而上唯物主义
		- 辩证唯物主义和历史唯物主义
	- 物质的根本属性是运动，运动是绝对的，静止是相对的
	- 时间和空间是运动着的物质的基本存在形式
	- 物质和意识的关系
		- 物质决定意识
		- 意识对物质具有反作用
		- 主观能动性和客观规律的辩证统一
		- 意识与人工智能
	- 什么是社会意识
		-  意识是知情意的统一体
		- 社会性是人的意识所固有的本质属性
		- 人类的自然语言是思维的物质外壳和意识的实现形式
	- <mark style="background: #FFF3A3A6;">世界的物质统一性</mark>是辩证唯物主义最基本，最核心的观点
		- 自然界是物质的
		- 人类社会本质上是物质的
		- 人的意识统一于物质
-  事物的发展规律和普遍联系
	-  主要内容是辩证法和形而上学
		- 唯物辩证法
			- 联系和发展的观点是唯物辩证法的总观点
				- 客观性
				- 普遍性
					- 任何事物的不同部分和要素都相互联系
					- 任何事物都不能孤立存在
					- 整个世界是相互联系的统一整体
				- 多样性
				- 条件性
			- 事物的变化发展特性
		- 形而上学
			- 采用孤立的，静止的观点看世界
		- 其焦点是是否承认事物的内在矛盾，是否承认矛盾是食物发展的根本原因
	- <mark style="background: #FFF3A3A6;">事物发展的一般规律</mark>
		- 对立统一规律(根本规律)
			- 揭示了事物普遍联系的根本内容和变化发展的动力
			- 对立统一规律是唯物辩证法的实质和核心
			- 提供了人们认识和改造世界的根本方法-矛盾分析法
				- 矛盾分析法是对立统一规律在方法论上的体现
				- 矛盾的同一性和斗争性是矛盾的两种基本属性
					- 矛盾的同一性
						- 概念
							- 同一性是矛盾的对立面相互依存，互为存在
							- 矛盾的对立面相互贯通，在一定条件下可以相互转化
							- 是有条件的，相对的
						- 表现
							- 同一性是事物存在和发展的前提
							- 同一性使矛盾双方相互吸收有利于自身的因素，在相互作用中推动各自发展
							- 同一性规定着事物转化的可能和发展的趋势
					- 斗争性
						- 概念
							- 矛盾着的对立面相互排斥，相互分离的性质和趋势
							- 是无条件的，绝对的
						- 表现
							- 矛盾双方斗争促进其力量发展的不平衡，促进对立面的转化和事物的质变
							- 矛盾双方的斗争是一种矛盾统一体向另一种矛盾统一体过渡的决定性力量
					- 矛盾的同一性和斗争性结合构成了事物的矛盾运动，推动事物的变化发展
				- 矛盾的普遍性和特殊性
					- 分析时需要坚持“两点论”和“重点论”的统一
					- 其原理是马克思主义基本原理同各国具体实际相结合的哲学基础
		- 质量互变规律
			- 事物包括质，量，度三个方面的规定性， 质是事物区别于其他事物的内在规定性，量是事物的规模，程度，速度等， 事物的量和质是统一的，量和质的统一在度中得到体现， 度是保持事物稳定性的数量和界限，即事物的限度，幅度，范围
			- 量变和质变的概念
				- 量变是事物数量的增减和组成要素排列次序的变动， 是保持事物的相对稳定性的不显著变化，体现了事物发展渐进过程的连续性
				- 质变是事物一种质态向另一种质态的飞跃， 体现了事物发展的渐进过程和连续性的中断。
			- 量变与质变的辩证关系
				- 量变是质变的必要准备
				- 质变是量变的必然结果， 并为新的量变开辟道路
				- 量变和质变是互相渗透的， 相互依存，相互贯通的，量变引起质变，交替循环构成事物的发展过程
		- <mark style="background: transparent; color: yellow">否定之否定规律</mark>
			- 唯物辩证法的否定内涵
				- 否定是事物的自我否定，自我发展，是事物内部矛盾运动的结果
				- 否定是事物的发展环节，是旧事物向新事物的转变，是旧质到新质的飞跃，只有经过否定，旧事物才能向新事物转变
				- 否定是新旧事物联系的环节，新旧事物是通过否定联系起来的
				- 辩证否定的实质是“扬弃”， 批判性地继承新事物， 克服消极因素，保留积极因素
			- <mark style="background:transparent ;color:yellow">事物需要经历肯定-否定-否定之否定三个发展阶段，是事物自我发展的两次否定过程</mark>
				- 第一次否定使矛盾得到初步解决，而处于否定阶段的事物仍然具有片面性
				- 需要经历否定之否定，以实现对立面的统一，使矛盾得到根本解决， 其中，“否定之否定”是向原来出发点的“回复”，但是是在更高阶段的回复
				- 否定之否定规律揭示了食物发展的前进性和曲折性的统一
					- 前进性体现在每一次否定都是质变，将事物推进到新阶段，每个周期都是开放的，前一个周期的终点时下一个周期的起点
					- 曲折性体现在回复性上，有暂时的停顿甚至是倒退，表明事物的发展规律不是直线式前进，而是螺旋式上升的
		- 联系和发展的基本环节-> 成对范畴
			- 内容与形式
			- 本质与现象
			- 原因与结果
			- 必然与偶然
			- 现实与可能
- 唯物辩证法
	- 唯物辩证法的本质和认识功能
		- 本质上是批判的和革命的
		- 马克思对唯物辩证法的本质解释
			- “辩证法对现存事物的肯定的理解中同时包含对现存事物的否定的理解， 即对现存事物必然灭亡的理解”
			- “辩证法对每一种既成的形式都是从不断的运动中，因而也是从其暂时性方面去理解”
			- “辩证法不崇拜任何东西，因而来说是批判的和革命的”
	- 唯物辩证法是客观辩证法和主观辩证法的统一
		- 唯物论和辩证法是统一的
		- 客观辩证法是指客观事物或者客观存在的辩证法
		- 主观辩证法是指人类认识和思维运动的辩证法
		- 唯物辩证法两者都包含，体现了唯物主义，辩证法和认识论的统一
	- 辩证思维方法和现代科学思维方法
		- 辩证思维方法
			- 主要有<mark style="background:transparent ;color:yellow">归纳演绎，分析综合，抽象和具体，逻辑和历史统一</mark>等等
				- 归纳演绎是人类思维从个别到一般，又从一般到个别的最常见思维，从事实中概括出一般性结论，由个别性前提过渡到一般性结论的推理形式，演绎是由一般性原则推导出个别性理论的推理形式,两者互为前提，相互补充
				- 分析与综合将认识的对象分解为各个部分，要素以便分别加以研究， 综合是将各个因素组合为一个整体的思维活动
				- 抽象与具体和分析与综合密切相关， 其思维方法是从具体到抽象再从抽象到具体的分析过程，同逻辑与历史相统一的方法有密切的关联。
				- 辩证法认为，逻辑和历史是一致的，因此历史的东西是逻辑事物的基础，逻辑的事物则是历史的东西在思维中的再现
		- 现代科学思维方法
			- 包括<u><mark style="background:transparent ;color:yellow">控制方法，信息方法，系统方法，模型方法和理想化方法等</mark></u>
			- 辩证思维方法和现代科学思维方法有着方法论上的共同性
				- 二者相互联系，相互补充。 
				- 辩证思维是现代科学思维的方法论前提
				- 现代科学思维方法需要自觉以辩证思维方法为指导，以创新方法系统
				- 现代科学思维方法丰富了辩证思维方法
		- 增强<u>辩证思维能力，历史思维能力，系统思维能力，战略思维能力，底线思维能力和创新思维能力</u>

#### 3. 实践认识及其发展规律

- 实践与认识
	- 科学实践观及其意义
		- 科学实践观的内涵
			- 强调全部的社会生活在本质上是实践的
			- 实践是检验真理的标准
			- 物质资料的生产是人类历史中首要的社会实践活动，是其他一切历史活动得以进行的前提
		- 科学实践观的意义
			- 克服了旧唯物主义的缺陷， 为辩证唯物主义奠定了科学的理论基础，在实践上将唯物论和辩证法有机统一起来
			- 建立了科学的，能动的，革命的反映论，实现了人类认识史上的变革
			- 在人类思想史上第一次揭示了社会生活的实践本质
			- 为人们能动地认识世界和改造世界提供了基本的思想方法和工作方法，强调理论付诸实践
	- <mark style="background:transparent ;color:yellow">实践的本质</mark>与基本结构
		- <mark style="background:transparent ;color:yellow">实践的本质</mark>
			- 实践具有客观实在性，自觉能动性和社会历史性三个基本特征
			- 实践具有自觉能动性，实践活动是一种有意识，有目的的活动
			- 实践具有社会历史性，实践是社会性的，历史性的活动 
				- 实践的社会性决定了其历史性
				- 实践的内容，性质，范围，水平都受一定的社会历史条件制约
		- 实践的基本结构
			- 实践是实践的主体和客体之间的相互作用
			- 实践的<u>主体，客体和中介</u>是实践活动的三项基本要素，三者的有机统一构成实践的基本结构
			- 实践的主体和客体相互作用的关系包括实践关系，认识关系和价值关系，其中实践关系是最根本的关系
		- 实践形式的多样性-> 基本类型
			- 一是物质生产实践
			- 二是社会政治实践
			- 三是科学文化实践
		- 实践对认识具有<u>决定性作用</u>
			- 实践是认识的来源
			- 实践是认识发展的动力
			- 实践是认识的目的
			- 实践是检验真理的唯一标准
	- <mark style="background:transparent ;color:yellow">认识的本质</mark>与发展规律
		- <mark style="background:indigo;color:yellow">认识的本质</mark>
			- 唯物主义
				- 认识的本质是从物到感觉和思想的认识路线
				- 坚持认识是人脑对客观世界的反映
					- 旧唯物主义认识论的特点是以感性直观为基础， 将人的认识看做是消极的， 被动地反映和接受外界的对象
				- <mark style="background: #FF5582A6;">辩证唯物主义对认识本质的科学回答</mark>
					- 继承了旧唯物主义的合理前提，认为认识是主体对客体的反映
					- 辩证唯物主义认识论有<u>两个突出特点</u>：
						- 把实践的观点作为整个认识论的基础
						- 把辩证法应用于反映论考察认识的发展过程
					- 内容
						- <mark style="background: transparent; color: yellow">认识的本质是主体在实践基础上对客体的能动反映，这种能动反映不仅具有反映客体内容的反映特征， 而且具有实践要求的主体能动的，创造性的特征。</mark>
						- <mark style="background: transparent; color: yellow">一方面，认识的反应特性是人类认识的基本规定性，另一方面，认识的能动反映具有创造性</mark>
			- 唯心主义(包括主观唯心主义和客观唯心主义)
				- 坚持从思想到感觉到物
				- 否认认识是人脑对客观世界的反映， 认为认识先于人的实践经验
				- 主观唯心主义认为人的认识是主观自生的， 是生而知之的
				- 客观唯心主义认为人的认识是上帝的启示或者某种客观精神的产物
			- 反映的能动性和创造性不可分割， 反映和创造是同一本质的两个不同方面
				- 创造离不开反映，反映离不开创造
	- <mark style="background: transparent; color: yellow">认识和实践</mark>的辩证关系
		- 从实践到认识
			- 感性认识和理性认识是人对客观世界的两种不同水平的反映形式， 是认识过程的两个不同阶段
				- 感性是在实践基础上对关于事物的现象，外部联系，事物各个方面的认识。包括感觉，知觉和表象三种形式
					- 从感觉，知觉到表象，是由个别的特性到完整的形象，感性认识具有不深刻的局限性
				- 理性认识是对事物的本质，全体，内部联系和事物自身规律的认识，包括概念，判断，推理三种形式
					- 从概念到判断到推理，是理性认识从低级到高级的发展
				- 理性认识和感性认识的关系
					- 理性认识依赖于感性认识
					- 感性认识有待于发展，深化为理性认识
						- 一、需要投身实践和深入调查
						- 二、需要经过思考的作用
					- 感性认识和理性认识相互渗透，相互包含
		- 从认识到实践(认识的第二次能动飞跃)
			- 意义性和重要性
				- 认识的目的是改造世界 
					- 将理论付诸实践， 是理论指导实践的过程，也是理论实现自身的过程
				- 认识的真理性只有在实践中才能得到检验和发展
					- 通过实践，正确的理论才能被证实，错误的理论才能被发现和推翻
		- 实践与认识的辩证运动及其规律
			- 实践与认识的辩证运动，是一个由感性到理性认识，又由理性认识到实践的飞跃，是实践，认识，飞跃的循环的过程
- 真理与价值的概念
	- <mark style="background: transparent; color: yellow">真理的概念</mark>
		- 真理是标志主观和客观相符合的哲学范畴，是对客观事物及规律的正确反映
		- 马克思主义真理观认为真理和客观事物之间的符合关系是建立在人类能动地改造客观世界的基础上，通过人的能动反映活动而实现，并且必然随着人类实践的发挥而拓展深化
	- 真理的客观性，绝对性和相对性
		- 客观性
			- 认识的目的和任务在于获得真理
			- 对于真理是不是客观的，一切唯物主义肯定，一切唯心主义否定
			- 真理的客观性指真理是是对客观事物及规律的正确反映， 其中包含不依赖于人和人的意识的客观内容， 客观性是真理的本质属性
			- 真理的客观性决定了真理的一元性
		- 绝对性和相对性
			- 真理的绝对性指真理的<mark style="background: transparent; color: yellow">主客观统一的确定性和发展的无限性</mark>
				- 一是任何真理都标志着主观和客观相符合， 包含不依赖于人和人的意识的客观内容
				- 二是人类认识按照本性来说，能够正确认识无限发展的物质世界， 这个意义上承认了世界的可知性，也承认了真理的绝对性
			- 真理的相对性指人们在一定条件下对客观事物及其本质的发展规律的正确认识总是有限度的，不完善的
				- 一是从客观世界整体来看，任何真理都只是对客观世界的某一阶段或某一部分的正确认识， 认识有待扩展
				- 就特定的事物而言，任何真理只是对客观对象的一定方面，层次和程度的正确认识，因而有待深化
			- 真理的绝对性和相对性相互依存，任何真理是绝对的也是相对的， 任何真理性的认识都是由相对性向绝对性转化过程中国的一个环节
			- 真理的绝对性和相对性根源于人类认识世界能力的无限性和有限性
			- 马克思主义作为客观真理，是绝对性和相对性的统一
		- 检验真理的标准
			- 实践是检验真理的唯一标准
				- 由真理的本性和实践的特点决定的
				- 真理的本质是客观性，本性在于主观和客观相符合
				- 实践具有直接性和现实性，具有直接实现性
				- 实践作为检验真理的标准，<mark style="background: transparent; color: yellow">既有确定性，也有不确定性(相对性)</mark>
	- 真理与<mark style="background: transparent; color: yellow">价值</mark>的辩证统一
		- <mark style="background: transparent; color: yellow">价值的概念</mark>
			- 价值是指实践基础上形成的主体和客体之间的意义关系，是客体对个人，群体乃至整个社会的生活和活动具有的积极意义
			- 在价值问题上，存在着客观主义价值论和主观主义价值论的对立，客观主义价值论认为价值是客体固有的，与主体无关。主观主义认为价值与客体无关
		- 价值的特性
			- 价值具有主体性，客观性 ，多维性和社会历史性，是价值本质的表现
				- 主体性
					- 价值关系的形成依赖于主题的存在
					- 价值关系的形成依赖于主体的创造
				- 客观性
					- 主体的存在和需要是客观的
					- 客体的存在，属性和作用是客观的
			- 价值的评价和主体直接相关， 其评价要以真理为依据，有科学和非科学之别
		- 实践活动受真理和价值尺度的制约，任何成功的实践，都是真理尺度和价值尺度的辩证统一
		- <mark style="background: transparent; color: yellow">价值观</mark>的概念
			- <mark style="background: transparent; color: yellow">价值观是人们关于价值本质的认识以及对人和事物的评价标准，评价原则和评价方法的观点的体系。</mark>
	- <mark style="background: transparent; color: yellow">认识世界和改造世界</mark>
		- 认识世界和改造世界的概念和基本特性
			- 其任务不仅在于解释世界，更重要的是在于为改造世界提供理论指导，实现主观客观，价值和实践的统一
			- 认识世界是主体能动地反映客体，获得关于事物的本质和发展规律和科学知识，探索啊掌握真理。改造世界是按照人类有利于自身的生存和发展的需要，改变事物的存在形式，创造自己理想世界和生活方式。
			- 认识世界和改造世界的关系：相互依赖，相互制约的辩证统一的关系
			- 主观和客观的矛盾是认识和改造世界的基本动力
			- 认识和改造世界的过程是从必然走向自由的过程
				- 自由是表示人的活动状态的范畴， 指人在活动中通过认识和利用必然表现出的一种自觉自主的状态
				- 自由有认识条件和实践条件
		- 马克思主义认识论
			- 一切从实际出发是马克思主义认识论的根本要求
			- 实事求是是中国共产党思想路线的核心
			- 理论创新和实践创新是相互促进，互相统一，良性互动的

#### 4. 人类社会的存在和发展

- 人类的社会存在和发展
	- 社会意识和社会存在
		- <mark style="background: transparent; color: yellow">社会历史观的基本问题，是社会存在和社会意识的关系问题</mark>, 这个问题是区别唯物史观和唯心史观的一个核心问题。唯物史观认为，<mark style="background: #FFF3A3A6;">社会存在决定社会意识</mark>
		- <mark style="background: transparent; color: yellow">社会存在</mark> 
			- <mark style="background: transparent; color: orange">社会存在是指社会物质生活条件， 是社会生活的物质方面，主要包括自然地理环境，人口因素和物质生产方式</mark>
				- 自然地理环境提供了社会生活和生产资料的来源
				- 人口因素是重要的物质社会物质生活条件
				- 物质生产方式是社会存在和发展的基础和决定力量
		- <mark style="background: transparent; color: yellow">社会意识</mark>
			- <mark style="background: transparent; color: orange">社会意识是社会存在的反映，是社会生活的精神方面。</mark>
				- 社会意识可以分为个体意识和群体意识
					- 个体意识是个人的生活经历和社会地位等在自己头脑中的反映
					- 群体意识是群体成员共同的社会意识形式
				- 社会意识可分为社会心理和社会意识形式
					- 社会心理是低层次的社会意识
					- 社会意识形式是高层次的社会意识，以理性认识为主
						- 社会意识形式以社会心理为基础，并对社会心理起指导和影响作用
						- 社会意识形式有意识形态和非意识形态之分
							- 意识形态是指反映一定经济关系，政治关系以及一定阶级或者社会集团利益和要求的思想体系，是社会意识诸形式中构成观念上层建筑的部分。
		- 社会存在和社会意识的辩证关系
			- 社会存在和社会意识是辩证统一的，<u>社会存在决定社会意识</u>，<u>社会意识是社会存在的反映</u>，社会意识产生的基础是人类的社会实践，实践的能动性决定了社会意识反映社会存在的能动性，所以<u>社会意识根源于社会存在，是对以实践为基础的不断变化发展的现实世界的反映</u>。
			- 社会意识既依赖于社会存在->在从根本上受到社会存在决定, 又有相对独立性
				- 社会意识和社会存在发展具有不完全同步性和不平衡性 
				- 社会意识内部各种形式之间存在相互影响且各自具有历史继承性
				- 社会意识相对于和会存在具有能动的反作用->通过指导人类实践来实现
	- <mark style="background: transparent; color: yellow">社会基本矛盾及其运动规律</mark>
		- 人类社会的基本矛盾: <mark style="background: transparent; color: orange">生产力与生产关系的矛盾，经济基础和上层建筑的矛盾</mark>，上述两种矛盾运动的规律，是人类社会发展的基本规律。
		- ==生产力==
			- <mark style="background: transparent; color: yellow">生产力是人类在生产实践中形成的改造和影响自然以使其适合社会需要的物质力量</mark>
			- 生产力的特性
				- 生产力具有<mark style="background: transparent; color: grey">客观现实性</mark>和<mark style="background: transparent; color: grey">社会历史性</mark>
				- 生产力的水平表现为<mark style="background: transparent; color: grey">生产发展的现实程度</mark>
				- 生产力的性质取决于<mark style="background: transparent; color: grey">生产的物质技术性质</mark>，主要是劳动资料的性质 
				- 生产力的状况是水平和性质的统一
			- 生产力的基本要素
				- <mark style="background: transparent; color: yellow">一是劳动资料(劳动手段)</mark>，其中最重要的是生产工具
				- <mark style="background: transparent; color: yellow">二是劳动对象</mark>
				- <mark style="background: transparent; color: yellow">三是劳动者</mark>
			- 科学技术是生产力中的重要因素，是先进生产力的几种体现的主要标志， 是第一生产力
		- <mark style="background: transparent; color: yellow">生产关系</mark>
			- 定义和概念
				- <mark style="background: transparent; color: orange">生产关系是人们在物质生产中形成的不以人意志为转移的经济关系</mark>
				- 生产关系是社会关系中最基本的关系，政治关系， 家庭关系，宗教关系等其他社会关系，都<u>受生产关系的支配和制约</u>
				- 生产关系包括<mark style="background: transparent; color: lime">生产资料所有制关系(最基本的，是进行物质资料生产前提)，生产中人与人的关系和产品的分配关系。</mark>
				- 生产关系适应所有制性质的要求时，对生产资料所有制起到巩固，发展的作用，否则会起到削弱，瓦解的作用
			- 类型
				- 生产关系区分为两种类型: <mark style="background: transparent; color: yellow">1. 以生产资料公有制为基础的生产关系，</mark>根本特征是<mark style="background: transparent; color: gray">生产资料为劳动者共同占有，人们在生产过程中处于平等地位</mark>   <mark style="background: transparent; color: yellow">2. 以生产资料私有制为基础的生产关系</mark>，根本特征是<mark style="background: transparent; color: gray">生产资料归少数剥削者占有，劳动者占有很少或没有生产资料，并在生产中处于被剥削地位</mark>
				- 以生产资料公有制为基础的生产关系必将取代以生产资料私有制为基础的生产关系
		- <mark style="background: transparent; color: orange">生产力是生产的物质内容， 生产关系是生产的社会形式，二者的有机统一构成社会的生产方式，其相互关系: 生产力决定生产关系，生产关系反作用于生产力</mark>
			- 生产力状况决定生产关系的性质，生产力的发展决定生产关系的变化
			- 当生产关系符合生产力发展的客观要求时，对生产力的发展起推动作用；当生产关系不适合生产力发展的客观要求时，就会阻碍生产力的发展
			- 生产力和生产关系的相互作用是一个过程，具体表现为二者的矛盾运动
		- 基本的<mark style="background: transparent; color: yellow">矛盾运动</mark>
			- <mark style="background: transparent; color: yellow">生产力和生产关系的矛盾运动</mark> 
				- 矛盾运动中内在的，本质的，必然的联系，就是生产关系一定要符合生产力状况的规律
					- <u>表现为生产关系对于生产力总是从基本适合到基本不相适合，再到新的基础上基本相适合， 生产关系也是相对稳定-> 新旧更替-> 相对稳定</u>
					- 生产力和生产关系的这种矛盾循环往复， 不断推动社会生产发展，进而推动整个社会逐步走向高级阶段
					- <mark style="background: transparent; color: lime">生产关系一定要适应生产力状况是社会形态发展的普遍规律</mark>
			- 经济基础和上层建筑的矛盾运动
				- 马克思将社会关系区分为经济基础和上层建筑两部分
				- <mark style="background: transparent; color: yellow">经济基础是由社会一定发展阶段的生产力所决定的生产关系的总和 </mark>
					- 理解经济基础的内涵的两点: <mark style="background: transparent; color: green">1. 社会的一定发展阶段上往往存在多种生产关系，2. 经济基础和经济体制有内在联系 </mark>
				- <mark style="background: transparent; color: yellow">上层建筑是建立在一定经济基础上的意识形态以及与之适应的制度，组织，措施</mark>
					-
