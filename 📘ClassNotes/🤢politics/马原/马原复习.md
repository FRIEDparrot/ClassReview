	- 基本方法：建立在辩证唯物主义和历史唯物主义世界观和方法论基础上，知道正确认识世界和改造世界的思想方法， 主要包括：

		-   群众路线的方法

		- 德国古典哲学
		- 英国古典政治经济学
		- 英法的空想社会主义

		- **科学性，人民性，实践性，发展性**

				- 联系的特点
					- 客观性
					- 普遍性
						- 任何事物的不同部分和要素都相互联系
						- 任何事物都不能孤立存在
						- 整个世界是相互联系的统一整体
					- 多样性
					- 条件性
			- 事物的变化发展特性
		- 形而上学
			- 采用孤立的，静止的观点看世界l
		- 其焦点是是否承认事物的内在矛盾，是否承认矛盾是事物发展的根本原因
	- <mark style="background: #FFF3A3A6;">事物发展的一般规律</mark>
		- <mark style="background: transparent; color: yellow">对立统一规律(根本规律)</mark>
			- <mark style="background: transparent; color: lime">对立统一规律是唯物辩证法的实质和核心</mark>
			- <mark style="background: transparent; color: lime">对立统一规律揭示了事物普遍联系的根本内容和变化发展的动力</mark>
			- 提供了人们认识和改造世界的根本方法-<mark style="background: transparent; color: yellow">矛盾分析法</mark>
				- 矛盾分析法是<mark style="background: transparent; color: yellow">对立统一规律在方法论上的体现</mark>
				- 矛盾的<mark style="background: transparent; color: yellow">同一性(统一属性)和斗争性(对立属性)</mark>是矛盾的两种基本属性
					- 矛盾的同一性
						- 概念
							- <mark style="background: transparent; color: yellow">矛盾的同一性是指矛盾着的对立面相互依存，相互贯通的性质和趋势</mark>
								- <mark style="background: transparent; color: gray">同一性是矛盾的对立面相互依存，互为存在的前提</mark>
								- <mark style="background: transparent; color: gray">矛盾的对立面相互贯通，在一定条件下可以相互转化</mark>
						- 矛盾的同一性在事物发展中的作用表现在：
							- <mark style="background: transparent; color: cyan">同一性是事物存在和发展的前提,发展中一方的发展以另一方的发展为条件，发展是在矛盾统一体内的发展</mark>
							- <mark style="background: transparent; color: cyan">同一性使矛盾双方相互吸收有利于自身的因素，在相互作用中推动各自发展</mark>
							- <mark style="background: transparent; color: cyan">同一性规定着事物转化的可能和发展的趋势，事物的发展方向和趋势不是随意的，而是有规律地向自己的对立面转化</mark>
					- 斗争性
						- <mark style="background: transparent; color: yellow">矛盾的斗争性是指矛盾着的对立面相互排斥， 相互分离的性质和趋势</mark>
							- <mark style="background: transparent; color: gray">矛盾的斗争可以分为对抗性矛盾和非对抗性矛盾两种基本形式</mark>
						- 矛盾斗争性<mark style="background: transparent; color: lime">在事物发展中的作用</mark>表现在：
							- 矛盾双方斗争<mark style="background: transparent; color: cyan">促进其力量发展的不平衡，促进对立面的转化和事物的质变</mark>
							- <mark style="background: transparent; color: cyan">矛盾双方的斗争是一种矛盾统一体向另一种矛盾统一体过渡的决定性力量， 矛盾双方排斥和否定促使旧的矛盾统一体破裂，新的矛盾统一体产生</mark>
					- <mark style="background: transparent; color: lime">矛盾的同一性和斗争性相互联结，相辅相成，相互依存， 斗争性寓于同一性之中，同一性通过斗争性体现出来， 同一性是有条件的，是相对的，斗争性是无条件的，绝对的。</mark>矛盾的同一性和斗争性结合构成了事物的矛盾运动，推动事物的变化发展
					- 在事物发展的过程中，矛盾的同一性和斗争性相互结合，共同发生作用，不同条件下二者所处的地位会有所不同
				- <mark style="background: transparent; color: yellow">矛盾的普遍性和特殊性</mark>及其相互关系
					- 矛盾的普遍性<mark style="background: transparent; color: yellow">是指矛盾存在于一切事物之中，存在于一切事物发展过程的始终，旧的矛盾解决了， 新的矛盾又产生，事物始终在矛盾中运动。</mark>
					- 矛盾的特殊性是指<mark style="background: transparent; color: yellow">各个具体事物的矛盾，每一个矛盾的各个方面在发展的不同阶段上各有其特点，矛盾的特殊性决定了事物的不同性质，只有具体分析矛盾的特殊性， 才能认清事物的本质和发展规律，并采取正确的方法和措施取解决矛盾， 推动事物的发展</mark>
					- 矛盾的普遍性和特殊性是辩证统一的关系， 矛盾的共性是无条件的，绝对的，矛盾的个性是有条件的，相对的， 任何现实存在的事物的矛盾都是共性和个性的有机统一
				- 主要矛盾和次要矛盾
					- <mark style="background: transparent; color: green">主要矛盾是矛盾体系中处于支配地位，对事物发展起决定性作用的矛盾， 次要矛盾是矛盾体系中处于从属地位，对事物发展起次要作用的矛盾。</mark> 每一对矛盾中， 处于支配地位，起主导作用的一方，是矛盾的主要方面, 处于从属地位，起次要作用的是矛盾的次要方面
					- 事物的性质是由主要矛盾的主要方面决定的
				- <mark style="background: transparent; color: yellow">把握两点论和重点论的统一</mark>
					- 分析时需要坚持“两点论”和“重点论”的统一
						- 两点论是指在分析事物的矛盾时，不仅要看到矛盾双方的对立，而且要看到矛盾双方的统一， 不仅要看到矛盾体系中存在主要矛盾和主要方面，还要看次要矛盾的次要方面
						- 重点论是指重点把握主要矛盾和矛盾的主要方面，并以此作为解决问题的出发点
					- 其原理是马克思主义基本原理同各国具体实际相结合的哲学基础
		- 质量互变规律
			- 事物包括质，量，度三个方面的规定性， 质是事物区别于其他事物的内在规定性，量是事物的规模，程度，速度等， 事物的量和质是统一的，量和质的统一在度中得到体现， 度是保持事物稳定性的数量和界限，即事物的限度，幅度，范围
			- 量变和质变的概念
				- 量变是事物数量的增减和组成要素排列次序的变动， 是保持事物的相对稳定性的不显著变化，体现了事物发展渐进过程的连续性
				- 质变是事物一种质态向另一种质态的飞跃， 体现了事物发展的渐进过程和连续性的中断。
			- 量变与质变的辩证关系
				- 量变是质变的必要准备
				- 质变是量变的必然结果， 并为新的量变开辟道路
				- 量变和质变是互相渗透的， 相互依存，相互贯通的，量变引起质变，交替循环构成事物的发展过程
		- <mark style="background: transparent; color: yellow">否定之否定规律</mark>
			- 唯物辩证法的否定内涵
				- 否定是事物的自我否定，自我发展，是事物内部矛盾运动的结果
				- 否定是事物的发展环节，是旧事物向新事物的转变，是旧质到新质的飞跃，只有经过否定，旧事物才能向新事物转变
				- 否定是新旧事物联系的环节，新旧事物是通过否定联系起来的
				- 辩证否定的实质是“扬弃”， 批判性地继承新事物， 克服消极因素，保留积极因素
			- <mark style="background:transparent ;color:yellow">事物需要经历肯定-否定-否定之否定三个发展阶段，是事物自我发展的两次否定过程</mark>
				- 第一次否定使矛盾得到初步解决，而处于否定阶段的事物仍然具有片面性
				- 需要经历否定之否定，以实现对立面的统一，使矛盾得到根本解决， 其中，“否定之否定”是向原来出发点的“回复”，但是是在更高阶段的回复
				- 否定之否定规律揭示了食物发展的前进性和曲折性的统一
					- 前进性体现在每一次否定都是质变，将事物推进到新阶段，每个周期都是开放的，前一个周期的终点时下一个周期的起点
					- 曲折性体现在回复性上，有暂时的停顿甚至是倒退，表明事物的发展规律不是直线式前进，而是螺旋式上升的
		- 联系和发展的基本环节-> 成对范畴
			- 内容与形式
			- 本质与现象
			- 原因与结果
			- 必然与偶然
			- 现实与可能
- 唯物辩证法
	- 唯物辩证法的本质和认识功能
		- 本质上是批判的和革命的
		- 马克思对唯物辩证法的本质解释
			- “辩证法对现存事物的肯定的理解中同时包含对现存事物的否定的理解， 即对现存事物必然灭亡的理解”
			- “辩证法对每一种既成的形式都是从不断的运动中，因而也是从其暂时性方面去理解”
			- “辩证法不崇拜任何东西，因而来说是批判的和革命的”
	- 唯物辩证法是客观辩证法和主观辩证法的统一
		- 唯物论和辩证法是统一的
		- 客观辩证法是指客观事物或者客观存在的辩证法
		- 主观辩证法是指人类认识和思维运动的辩证法
		- 唯物辩证法两者都包含，体现了唯物主义，辩证法和认识论的统一
	- 辩证思维方法和现代科学思维方法
		- 辩证思维方法
			- 主要有<mark style="background:transparent ;color:yellow">归纳演绎，分析综合，抽象和具体，逻辑和历史统一</mark>等等
				- 归纳演绎是人类思维从个别到一般，又从一般到个别的最常见思维，从事实中概括出一般性结论，由个别性前提过渡到一般性结论的推理形式，演绎是由一般性原则推导出个别性理论的推理形式,两者互为前提，相互补充
				- 分析与综合将认识的对象分解为各个部分，要素以便分别加以研究， 综合是将各个因素组合为一个整体的思维活动
				- 抽象与具体和分析与综合密切相关， 其思维方法是从具体到抽象再从抽象到具体的分析过程，同逻辑与历史相统一的方法有密切的关联。
				- 辩证法认为，逻辑和历史是一致的，因此历史的东西是逻辑事物的基础，逻辑的事物则是历史的东西在思维中的再现
		- 现代科学思维方法
			- 包括<u><mark style="background:transparent ;color:yellow">控制方法，信息方法，系统方法，模型方法和理想化方法等</mark></u>
			- 辩证思维方法和现代科学思维方法有着方法论上的共同性
				- 二者相互联系，相互补充。 
				- 辩证思维是现代科学思维的方法论前提
				- 现代科学思维方法需要自觉以辩证思维方法为指导，以创新方法系统
				- 现代科学思维方法丰富了辩证思维方法
		- 增强<u>辩证思维能力，历史思维能力，系统思维能力，战略思维能力，底线思维能力和创新思维能力</u>

#### 3. 实践认识及其发展规律

- 实践与认识
	- 科学实践观及其意义
		- 科学实践观的内涵
			- 强调全部的社会生活在本质上是实践的
			- 实践是检验真理的标准
			- 物质资料的生产是人类历史中首要的社会实践活动，是其他一切历史活动得以进行的前提
		- 科学实践观的意义
			- 克服了旧唯物主义的缺陷， 为辩证唯物主义奠定了科学的理论基础，在实践上将唯物论和辩证法有机统一起来
			- 建立了科学的，能动的，革命的反映论，实现了人类认识史上的变革
			- 在人类思想史上第一次揭示了社会生活的实践本质
			- 为人们能动地认识世界和改造世界提供了基本的思想方法和工作方法，强调理论付诸实践
	- <mark style="background:transparent ;color:yellow">实践的本质</mark>与基本结构
		- <mark style="background:transparent ;color:yellow">实践的本质</mark>
			- 实践具有客观实在性，自觉能动性和社会历史性三个基本特征
			- 实践具有自觉能动性，实践活动是一种有意识，有目的的活动
			- 实践具有社会历史性，实践是社会性的，历史性的活动 
				- 实践的社会性决定了其历史性
				- 实践的内容，性质，范围，水平都受一定的社会历史条件制约
		- 实践的基本结构
			- 实践是实践的主体和客体之间的相互作用
			- 实践的<u>主体，客体和中介</u>是实践活动的三项基本要素，三者的有机统一构成实践的基本结构
			- 实践的主体和客体相互作用的关系包括实践关系，认识关系和价值关系，其中实践关系是最根本的关系
		- 实践形式的多样性-> 基本类型
			- 一是物质生产实践
			- 二是社会政治实践
			- 三是科学文化实践
		- 实践对认识具有<u>决定性作用</u>
			- 实践是认识的来源
			- 实践是认识发展的动力
			- 实践是认识的目的
			- 实践是检验真理的唯一标准
	- <mark style="background:transparent ;color:yellow">认识的本质</mark>与发展规律
		- <mark style="background:indigo;color:yellow">认识的本质</mark>
			- 唯物主义
				- 认识的本质是从物到感觉和思想的认识路线
				- 坚持认识是人脑对客观世界的反映
					- 旧唯物主义认识论的特点是以感性直观为基础， 将人的认识看做是消极的， 被动地反映和接受外界的对象
				- <mark style="background: #FF5582A6;">辩证唯物主义对认识本质的科学回答</mark>
					- 继承了旧唯物主义的合理前提，认为认识是主体对客体的反映
					- 辩证唯物主义认识论有<u>两个突出特点</u>：
						- 把实践的观点作为整个认识论的基础
						- 把辩证法应用于反映论考察认识的发展过程
					- 内容
						- <mark style="background: transparent; color: yellow">认识的本质是主体在实践基础上对客体的能动反映，这种能动反映不仅具有反映客体内容的反映特征， 而且具有实践要求的主体能动的，创造性的特征。</mark>
						- <mark style="background: transparent; color: yellow">一方面，认识的反应特性是人类认识的基本规定性，另一方面，认识的能动反映具有创造性</mark>
			- 唯心主义(包括主观唯心主义和客观唯心主义)
				- 坚持从思想到感觉到物
				- 否认认识是人脑对客观世界的反映， 认为认识先于人的实践经验
				- 主观唯心主义认为人的认识是主观自生的， 是生而知之的
				- 客观唯心主义认为人的认识是上帝的启示或者某种客观精神的产物
			- 反映的能动性和创造性不可分割， 反映和创造是同一本质的两个不同方面
				- 创造离不开反映，反映离不开创造
	- <mark style="background: transparent; color: yellow">认识和实践</mark>的辩证关系
		- 从实践到认识
			- 感性认识和理性认识是人对客观世界的两种不同水平的反映形式， 是认识过程的两个不同阶段
				- 感性是在实践基础上对关于事物的现象，外部联系，事物各个方面的认识。包括感觉，知觉和表象三种形式
					- 从感觉，知觉到表象，是由个别的特性到完整的形象，感性认识具有不深刻的局限性
				- 理性认识是对事物的本质，全体，内部联系和事物自身规律的认识，包括概念，判断，推理三种形式
					- 从概念到判断到推理，是理性认识从低级到高级的发展
				- 理性认识和感性认识的关系
					- 理性认识依赖于感性认识
					- 感性认识有待于发展，深化为理性认识
						- 一、需要投身实践和深入调查
						- 二、需要经过思考的作用
					- 感性认识和理性认识相互渗透，相互包含
		- 从认识到实践(认识的第二次能动飞跃)
			- 意义性和重要性
				- 认识的目的是改造世界 
					- 将理论付诸实践， 是理论指导实践的过程，也是理论实现自身的过程
				- 认识的真理性只有在实践中才能得到检验和发展
					- 通过实践，正确的理论才能被证实，错误的理论才能被发现和推翻
		- 实践与认识的辩证运动及其规律
			- 实践与认识的辩证运动，是一个由感性到理性认识，又由理性认识到实践的飞跃，是实践，认识，飞跃的循环的过程
- 真理与价值的概念
	- <mark style="background: transparent; color: yellow">真理的概念</mark>
		- 真理是标志主观和客观相符合的哲学范畴，是对客观事物及规律的正确反映
		- 马克思主义真理观认为真理和客观事物之间的符合关系是建立在人类能动地改造客观世界的基础上，通过人的能动反映活动而实现，并且必然随着人类实践的发挥而拓展深化
	- 真理的客观性，绝对性和相对性
		- 客观性
			- 认识的目的和任务在于获得真理
			- 对于真理是不是客观的，一切唯物主义肯定，一切唯心主义否定
			- 真理的客观性指真理是是对客观事物及规律的正确反映， 其中包含不依赖于人和人的意识的客观内容， 客观性是真理的本质属性
			- 真理的客观性决定了真理的一元性
		- 绝对性和相对性
			- 真理的绝对性指真理的<mark style="background: transparent; color: yellow">主客观统一的确定性和发展的无限性</mark>
				- 一是任何真理都标志着主观和客观相符合， 包含不依赖于人和人的意识的客观内容
				- 二是人类认识按照本性来说，能够正确认识无限发展的物质世界， 这个意义上承认了世界的可知性，也承认了真理的绝对性
			- 真理的相对性指人们在一定条件下对客观事物及其本质的发展规律的正确认识总是有限度的，不完善的
				- 一是从客观世界整体来看，任何真理都只是对客观世界的某一阶段或某一部分的正确认识， 认识有待扩展
				- 就特定的事物而言，任何真理只是对客观对象的一定方面，层次和程度的正确认识，因而有待深化
			- 真理的绝对性和相对性相互依存，任何真理是绝对的也是相对的， 任何真理性的认识都是由相对性向绝对性转化过程中国的一个环节
			- 真理的绝对性和相对性根源于人类认识世界能力的无限性和有限性
			- 马克思主义作为客观真理，是绝对性和相对性的统一
		- 检验真理的标准
			- 实践是检验真理的唯一标准
				- 由真理的本性和实践的特点决定的
				- 真理的本质是客观性，本性在于主观和客观相符合
				- 实践具有直接性和现实性，具有直接实现性
				- 实践作为检验真理的标准，<mark style="background: transparent; color: yellow">既有确定性，也有不确定性(相对性)</mark>
	- 真理与<mark style="background: transparent; color: yellow">价值</mark>的辩证统一
		- <mark style="background: transparent; color: yellow">价值的概念</mark>
			- 价值是指实践基础上形成的主体和客体之间的意义关系，是客体对个人，群体乃至整个社会的生活和活动具有的积极意义
			- 在价值问题上，存在着客观主义价值论和主观主义价值论的对立，客观主义价值论认为价值是客体固有的，与主体无关。主观主义认为价值与客体无关
		- 价值的特性
			- 价值具有主体性，客观性 ，多维性和社会历史性，是价值本质的表现
				- 主体性
					- 价值关系的形成依赖于主题的存在
					- 价值关系的形成依赖于主体的创造
				- 客观性
					- 主体的存在和需要是客观的
					- 客体的存在，属性和作用是客观的
			- 价值的评价和主体直接相关， 其评价要以真理为依据，有科学和非科学之别
		- 实践活动受真理和价值尺度的制约，任何成功的实践，都是真理尺度和价值尺度的辩证统一
		- <mark style="background: transparent; color: yellow">价值观</mark>的概念
			- <mark style="background: transparent; color: yellow">价值观是人们关于价值本质的认识以及对人和事物的评价标准，评价原则和评价方法的观点的体系。</mark>
	- <mark style="background: transparent; color: yellow">认识世界和改造世界</mark>
		- 认识世界和改造世界的概念和基本特性
			- 其任务不仅在于解释世界，更重要的是在于为改造世界提供理论指导，实现主观客观，价值和实践的统一
			- 认识世界是主体能动地反映客体，获得关于事物的本质和发展规律和科学知识，探索和掌握真理。改造世界是按照人类有利于自身的生存和发展的需要，改变事物的存在形式，创造自己理想世界和生活方式。
			- 认识世界和改造世界的关系：相互依赖，相互制约的辩证统一的关系
			- 主观和客观的矛盾是认识和改造世界的基本动力
			- 认识和改造世界的过程是从必然走向自由的过程
				- 自由是表示人的活动状态的范畴， 指人在活动中通过认识和利用必然表现出的一种自觉自主的状态
				- 自由有认识条件和实践条件
		- 马克思主义认识论
			- 一切从实际出发是马克思主义认识论的根本要求
			- 实事求是是中国共产党思想路线的核心
			- 理论创新和实践创新是相互促进，互相统一，良性互动的

#### 4. 人类社会的存在和发展

注：以下的三点是唯物史观和唯心史观的主要区别 
- 人类的社会存在和发展
	- 社会意识和社会存在
		- <mark style="background: transparent; color: yellow">社会历史观的基本问题，是社会存在和社会意识的关系问题</mark>, 这个问题是区别唯物史观和唯心史观的一个核心问题。唯物史观认为，<mark style="background: #FFF3A3A6;">社会存在决定社会意识</mark>
		- <mark style="background: transparent; color: yellow">社会存在</mark> 
			- <mark style="background: transparent; color: orange">社会存在是指社会物质生活条件， 是社会生活的物质方面，主要包括自然地理环境，人口因素和物质生产方式</mark>
				- 自然地理环境提供了社会生活和生产资料的来源
				- 人口因素是重要的物质社会物质生活条件
				- 物质生产方式是社会存在和发展的基础和决定力量
		- <mark style="background: transparent; color: yellow">社会意识</mark>
			- <mark style="background: transparent; color: orange">社会意识是社会存在的反映，是社会生活的精神方面。</mark>
				- 社会意识可以分为个体意识和群体意识
					- 个体意识是个人的生活经历和社会地位等在自己头脑中的反映
					- 群体意识是群体成员共同的社会意识形式
				- 社会意识可分为社会心理和社会意识形式
					- 社会心理是低层次的社会意识
					- 社会意识形式是高层次的社会意识，以理性认识为主
						- 社会意识形式以社会心理为基础，并对社会心理起指导和影响作用
						- 社会意识形式有意识形态和非意识形态之分
							- 意识形态是指反映一定经济关系，政治关系以及一定阶级或者社会集团利益和要求的思想体系，是社会意识诸形式中构成观念上层建筑的部分。
		- 社会存在和社会意识的辩证关系
			- 社会存在和社会意识是辩证统一的，<u>社会存在决定社会意识</u>，<u>社会意识是社会存在的反映</u>，社会意识产生的基础是人类的社会实践，实践的能动性决定了社会意识反映社会存在的能动性，所以<u>社会意识根源于社会存在，是对以实践为基础的不断变化发展的现实世界的反映</u>。
			- 社会意识既依赖于社会存在->在从根本上受到社会存在决定, 又有相对独立性
				- 社会意识和社会存在发展具有不完全同步性和不平衡性 
				- 社会意识内部各种形式之间存在相互影响且各自具有历史继承性
				- 社会意识相对于和会存在具有能动的反作用->通过指导人类实践来实现
		- 文化是社会意识的重要组成部分， 其对社会发展的重要作用主要表现在
			- 文化为社会发展提供思想保证
			- 文化为社会发展提供精神动力
			- 文化为社会发展提供智力支持
			- 文化为社会发展凝聚力量
	- <mark style="background: transparent; color: yellow">社会基本矛盾及其运动规律</mark>
		- 人类社会的基本矛盾: <mark style="background: transparent; color: orange">生产力与生产关系的矛盾，经济基础和上层建筑的矛盾</mark>，上述两种矛盾运动的规律，是人类社会发展的基本规律。
		- ==生产力==
			- <mark style="background: transparent; color: yellow">生产力是人类在生产实践中形成的改造和影响自然以使其适合社会需要的物质力量</mark>
			- 生产力的特性
				- 生产力具有<mark style="background: transparent; color: grey">客观现实性</mark>和<mark style="background: transparent; color: grey">社会历史性</mark>
				- 生产力的水平表现为<mark style="background: transparent; color: grey">生产发展的现实程度</mark>
				- 生产力的性质取决于<mark style="background: transparent; color: grey">生产的物质技术性质</mark>，主要是劳动资料的性质 
				- 生产力的状况是水平和性质的统一
			- 生产力的基本要素
				- <mark style="background: transparent; color: yellow">一是劳动资料(劳动手段)</mark>，其中最重要的是生产工具
				- <mark style="background: transparent; color: yellow">二是劳动对象</mark>
				- <mark style="background: transparent; color: yellow">三是劳动者</mark>
			- 科学技术是生产力中的重要因素，是先进生产力的几种体现的主要标志， 是第一生产力
		- <mark style="background: transparent; color: yellow">生产关系</mark>
			- 定义和概念
				- <mark style="background: transparent; color: orange">生产关系是人们在物质生产中形成的不以人意志为转移的经济关系</mark>
				- 生产关系是社会关系中最基本的关系，政治关系， 家庭关系，宗教关系等其他社会关系，都<u>受生产关系的支配和制约</u>
				- 生产关系包括<mark style="background: transparent; color: lime">生产资料所有制关系(最基本的，是进行物质资料生产前提)，生产中人与人的关系和产品的分配关系。</mark>
				- 生产关系适应所有制性质的要求时，对生产资料所有制起到巩固，发展的作用，否则会起到削弱，瓦解的作用
			- 类型
				- 生产关系区分为两种类型: <mark style="background: transparent; color: yellow">1. 以生产资料公有制为基础的生产关系，</mark>根本特征是<mark style="background: transparent; color: gray">生产资料为劳动者共同占有，人们在生产过程中处于平等地位</mark>   <mark style="background: transparent; color: yellow">2. 以生产资料私有制为基础的生产关系</mark>，根本特征是<mark style="background: transparent; color: gray">生产资料归少数剥削者占有，劳动者占有很少或没有生产资料，并在生产中处于被剥削地位</mark>
				- 以生产资料公有制为基础的生产关系必将取代以生产资料私有制为基础的生产关系
		- <mark style="background: transparent; color: orange">生产力和生产关系是社会生产不可分割的两个方面: 生产力是生产的物质内容， 生产关系是生产的社会形式，二者的有机统一构成社会的生产方式，其相互关系: 生产力决定生产关系，生产关系反作用于生产力</mark>
			- 生产力状况决定生产关系的性质，生产力的发展决定生产关系的变化
			- 当生产关系符合生产力发展的客观要求时，对生产力的发展起推动作用；当生产关系不适合生产力发展的客观要求时，就会阻碍生产力的发展
			- 生产力和生产关系的相互作用是一个过程，具体表现为二者的矛盾运动
		- 基本的<mark style="background: transparent; color: yellow">矛盾运动</mark>
			- <mark style="background: transparent; color: yellow">生产力和生产关系的矛盾运动</mark> 
				- 矛盾运动中<u>内在的，本质的，必然的联系，就是生产关系一定要符合生产力状况的规律</u>
					- <u>表现为生产关系对于生产力总是从基本适合到基本不相适合，再到新的基础上基本相适合， 生产关系也是相对稳定-> 新旧更替-> 相对稳定</u>
					- 生产力和生产关系的这种矛盾循环往复， 不断推动社会生产发展，进而推动整个社会逐步走向高级阶段
					- <mark style="background: transparent; color: lime">生产关系一定要适应生产力状况是社会形态发展的普遍规律</mark>
				- 生产力与生产关系矛盾运动的理论和现实意义
					- 在人类历史上彻底否定了单纯以道德为评判历史功过是非标准的思想体系， 第一次科学地确立了生产力发展是“社会进步的最高标准”，并将<u>生产力和生产关系的矛盾运动的规律作为判断时代变革的客观依据</u>
					- 生产力和生产关系的矛盾运动规律是马克思主义政党制定路线，方针和政策的重要依据
			- 经济基础和上层建筑的矛盾运动
				- 马克思将社会关系区分为经济基础和上层建筑两部分
				- <mark style="background: transparent; color: yellow">经济基础是由社会一定发展阶段的生产力所决定的生产关系的总和 </mark>
					- 理解经济基础的内涵的两点: <mark style="background: transparent; color: green">1. 社会的一定发展阶段上往往存在多种生产关系，但决定一个社会性质的是其中占支配地位的关系 2. 经济基础和经济体制有内在联系 </mark>
				- <mark style="background: transparent; color: yellow">上层建筑是建立在一定经济基础上的意识形态以及与之适应的制度，组织，设施</mark>
					- 上层建筑由意识形态以及政治法律制度及设施和政治组织两部分构成
						- 意识形态又称观念上层建筑，包括政治法律思想，道德，艺术，宗教，哲学等等
						- 政治法律制度以及设施和政治组织又称政治上层建筑， 居于主导地位，国家政权是政治上层建筑的核心
					- 马克思将国家分为国体和政体两个方面
						- 国体指<mark style="background: transparent; color: yellow">社会各阶级在国家中的地位， 表明国家政权掌握在哪个阶级手里， 哪个阶级是统治阶级， 哪个阶级是被统治阶级</mark>
						- <mark style="background: transparent; color: yellow">政体指统治阶级实现阶级统治的具体组织形式， 即政权构成形式</mark>
				- 经济基础和上层建筑是辩证统一的
					- <mark style="background: transparent; color: yellow">经济基础决定上层建筑，上层建筑反作用于经济基础，相互影响，相互作用</mark>
						- <mark style="background: transparent; color: orange">经济基础是上层建筑产生，存在和发展的物质基础。上层建筑是经济基础得以确立统治地位并获得巩固发展不可缺少的政治，思想条件。经济基础的性质决定上层建筑的性质 。经济基础的变更必然引起上层建筑的变革</mark>， 并决定变革的方向
						- 上层建筑对经济基础反作用的集中体现: <mark style="background: transparent; color: orange">上层建筑为自己的经济基础的形成和巩固服务， 确立或维护其在社会中的统治地位</mark>
					- 经济基础和上层建筑的<mark style="background: transparent; color: skyblue">相互作用构成二者的矛盾运动</mark>
						- <mark style="background: transparent; color: orange">在同一性质的经济基础和上层建筑的关系中，上层建筑的不完善部分， 没有反映经济基础要求的部分都会同经济基础发生矛盾</mark>
						- <mark style="background: transparent; color: orange">在不同性质的经济基础与上层建筑的残余， 未来上层建筑的萌芽之间的矛盾。</mark>
						- <mark style="background: transparent; color: orange">当一种社会形态处于上升发展阶段时， 上层建筑对于经济基础一般是适应的， 当一种社会形态处于没落时期时， 上层建筑同经济基础的变革的客观要求是不相适应的， 其矛盾则变为对抗性的，全局性的矛盾</mark>
	- 人类的普遍交往和世界历史的形成与发展
		- 交往的概念
			- 交往指在一定条件下，现实的个人，群体，阶级，民族，国家之间在物质和精神上相互往来，相互作用， 彼此联系的活动
			- 交往可以分为物质交往和精神交往
			- 交往是人类实践获得感的重要组成部分，对社会生活有重要的影响
				- 促进生产力的发展
				- 促进社会关系的进步
				- 促进文化的发展与传播
				- 促进人的全面发展
		- 世界历史的形成和发展
			- 人类历史向世界历史转变是资本主义生产方式出现和享世界扩张的结果，世界历史的形成又反过来促进了生产力的普遍发展和人类的普遍交往， 推动社会发展， 为人的发展创造条件
				- <mark style="background: transparent; color: yellow">生产方式的变革是世界历史形成和发展的基础</mark>
				- 普遍交往是世界历史的而基本特征
	- <mark style="background: transparent; color: red">社会形态更替的一般规律以及特殊形式</mark>
		- 社会形态的内涵
			- <mark style="background: transparent; color: lime">社会形态是关于社会运动的具体形式，发展阶段和不同质态的范畴，是同生产力发展一定阶段相适应的经济基础与上层建筑的统一体</mark>
			- 社会形态包括社会的<mark style="background: #FFF3A3A6;">经济形态，政治形态</mark>和<mark style="background: #FFF3A3A6;">意识形态</mark>，是三者具体的，历史的统一
				- 经济形态是社会形态的基础，生产资料所有制关系具有决定性的意义。
				- 建立在经济基础上的上层建筑则是社会形体不可分割的组成部分。一定的社会形态总要以一定的社会制度呈现出来
				- 社会制度能够几种体现社会形态的性质
		- 社会形态的统一性和多样性
			- 从原始社会，奴隶，封建，资本主义，共产主义社会(社会主义是第一阶段)这五种社会形态的依次更替，是社会历史运动的一般过程的一般规律，表现了社会形态更替的统一性
			- 不同的国家民族的社会发展历程不同，同时即使是同一种社会形态，不同的国家发展显现出特点也不同，表现了多样性
		- 社会形态更替的必然性
			- 社会形态更替的统一性和多样性根源于社会形态更替的<mark style="background: transparent; color: yellow">客观必然性</mark>与人们的历史选择性想统一的过程。 
			- <mark style="background: transparent; color: orange">社会形态更替的客观必然性主要是指社会形态依次更替的过程和规律是客观的，发展的基本趋势是确定不移的。</mark><mark style="background: transparent; color: cyan">社会形态更替归根结底是社会基本矛盾运动的结果， 其中，生产力的发展具有最终的决定意义</mark>
			- 社会更替具有前进性和曲折性
- 社会历史的发展的动力
	- 概念
		- 唯心主义将社会和历史的发展动力归结为人们的思想动机或者精神力量，未能揭示社会历史发展的本质
		- 唯物主义的观点: 物质生产方式是社会发展的基础，在此基础上形成的<mark style="background: #FF5582A6;">生产力和生产关系的矛盾， 经济基础和上层建筑的矛盾是社会发展的基本矛盾和根本动力</mark>
	- <mark style="background: transparent; color: yellow">经济基础和上层建筑的矛盾</mark>是社会基本矛盾
		- 社会基本矛盾是指<u>贯穿社会发过程始终， 规定社会发展过程的基本性质和基本趋势，并对社会历史发展起根本推动作用的矛盾</u>
		- 上述两种基本矛盾规定并反映了社会基本结构的性质和基本面貌
	- 社会基本矛盾和社会主要矛盾
		- <mark style="background: transparent; color: yellow">社会基本矛盾是历史发展的根本动力</mark>
			- 主要表现
				- 1 <mark style="background: transparent; color: violet">生产力是社会基本矛盾中最基本的动力因素，是人类社会发展进步的最终决定力量</mark>
					- 生产力决定生产关系的性质，进而决定其他社会关系的基本面貌， 决定世界历史的发展进程
					- 作为<mark style="background: transparent; color: yellow">社会历史发展基础的物质生产</mark>存在着双重关系，体现为<mark style="background: transparent; color: lime">生产力的人与自然的关系以及生产关系中的人与人的关系，双重关系构成了社会发展过程中的最基本的矛盾</mark>
				- 2 <mark style="background: transparent; color: violet">社会基本矛盾特别是生产力和生产关系的矛盾，决定着社会中其他矛盾的存在和发展</mark>
					- 生产力和生产关系的矛盾，决定经济基础和上层建筑的矛盾的产生和发展。
					- 经济基础和上层建筑的矛盾，也会影响和制约生产力和生产关系的矛盾
				- 3 <mark style="background: transparent; color: violet">社会矛盾具有不同的表现形式和解决方式，并从根本上影响和促进社会形态的变化和发展</mark>
					- 阶级社会中，社会基本矛盾往往通过一定的阶层或阶级的矛盾表现出来，或者表现为不同社会集团之间的利益和冲突
					- 同一社会形态的发展中，<mark style="background: transparent; color: teal">社会基本矛盾往往是通过改革解决的，成功的改革能解决社会的基本矛盾</mark>而当社会基本矛盾尖锐化时， 会导致不同的生产关系和政治法律制度的阶级斗争甚至社会革命， 往往促成一定社会形态的变迁和更替
		- 社会<mark style="background: transparent; color: yellow">主要矛盾</mark>在历史发展中的作用
			- <mark style="background: transparent; color: red">社会基本矛盾是其他一切社会矛盾的根源，规定制约着社会主要矛盾的存在和发展</mark>
				- <mark style="background: transparent; color: lime">社会主要矛盾是社会基本矛盾的具体体现</mark>
				- 社会基本矛盾即生产力和生产关系， 经济基础和上层建筑的矛盾，它们贯穿并制约着社会的发展的全过程， 规定社会发展过程的基本性质 
			-  <mark style="background: transparent; color: red">社会主要矛盾是处于支配地位， 在社会发展过程一定阶段上起主导作用的矛盾。</mark>
				- 存在主要矛盾和非主要矛盾的区别，社会主要矛盾的存在和发展， 规定影响着社会非主要矛盾的存在和发展
				- 由于社会经济， 政治，文化因素的变化，原有的社会主要矛盾会朝着两个方向转化， 一是社会主要矛盾双方的内容发生一定的变化，二是矛盾地位发生变化(社会主要矛盾和非主要矛盾在一定条件下可以相互转化)。社会主要矛盾的变化会影响社会发展的过程， 主要表现为社会发展过程中出现了新的阶段性特点。
			- 我国现阶段的社会主要矛盾是<mark style="background: transparent; color: yellow">人民日益增长的美好文化需要和不平衡不充分发展之间的矛盾(p142)</mark>
				- 这一新时代的主要矛盾的形成和出现，是中国特色社会主义进入新时代的重要标志和依据
	- 阶级和阶级斗争
		- 阶级斗争和社会革命在阶级社会发展中的作用
			- 阶级和阶级斗争是社会发展的直接动力 
			- 其推动作用更突出地表现在社会形态的更替中
			- 阶级斗争及其作用受到一定历史条件的制约
		- 认识阶级的科学方法

		- 社会革命的实质和作用
			- 社会革命有广义和狭义之分，既是社会基本矛盾运动的结果，也是推动社会发展特别是社会形态更替的重要动力
			- 革命对推动社会发展起巨大作用
				- 首先，革命是实现社会形态更替的重要手段和决定性环节。
				- 社会的革命能充分发挥人民群众创造历史的积极性和伟大作用
				- 社会革命能极大地教育锻炼包括革命阶级在内的广大人民群众
				- 无产阶级革命将为消除阶级对抗，并充分利用全人类文明成果促进社会全面进步创造条件
		- 改革在社会发展中的作用
			- 是在一定程度上解决社会矛盾，促进生产力发展，推动社会进步的有效途径和手段
		- 科技在社会发展中的作用

			- 科技是把双刃剑，需要正确认识和把握科学技术的社会作用

#### 5. 资本主义的运作及其规律
- 资本主义的本质及规律
	- 商品经济及价值规律
		- 商品经济的概念，形成与发展
			- 商品经济的发展经历了简单商品经济，发达商品经济两个阶段， <u>简单商品经济以生产资料私有制和个体劳动为基础，资本主义商品经济以生产资料私有制和雇佣劳动为基础</u>
			- 商品经济产生的历史条件
				-<mark style="background: transparent; color: yellow"> 商品经济是以交换为目的而进行生产的经济形式</mark>， 是一定社会历史条件的产物。
				- 商品经济得以产生的<mark style="background: #FFF3A3A6;">两个历史条件</mark>：
					- 1 <mark style="background: transparent; color: cyan">存在社会分工</mark>
					- 2 <mark style="background: transparent; color: cyan">生产资料和劳动产品属于不同的所有者</mark>
				- 商品经济作为自然经济的对立物产生和发展起来
					- 自然经济下的社会由许多分散单一的经济单位组成，每个经济单位从事各种经济工作
					- 生产力水平低下，排斥社会分工，生产规模狭小
		- 商品的基本属性和劳动的二重性
			- <mark style="background: transparent; color: lime">商品是用来交换，能满足人的某种需要的劳动产品，具有使用价值和价值两个因素或者两种属性，称为商品的二因素</mark>
				- 商品的使用价值是通过交换用来满足他人和社会的需要的。是交换价值的物质承担者
				- 决定商品交换的比例的不是商品的使用价值，而是价值
				- 价值是凝结在商品中的无差别的一般人类劳动，即人的脑力和体力耗费
			- <mark style="background: transparent; color: orange">价值是商品特有的社会属性</mark>
				- 商品交换实际上是商品生产者之间相互交换劳动的关系，商品的价值在本质上体现了生产者之间一定的社会关系
			- <mark style="background: transparent; color: violet">生产商品的劳动具有二重性， 即具体劳动和抽象劳动。</mark> 
				- <mark style="background: transparent; color: orange">具体劳动是指生产一定使用价值的具体形式的劳动</mark>
				- <mark style="background: transparent; color: orange">抽象劳动是指撇开一切具体形式的，无差别的一般人类劳动，即人的体力劳动和脑力的耗费。</mark>
				- 任何一种生产商品的劳动，一方面是特殊的具体劳动，另一方面是一般的抽象劳动。 劳动的二重性决定了商品的二因素。 具体劳动和抽象劳动也是对立统一的关系
			- 商品价值量的决定
				- 商品的价值包括<mark style="background: transparent; color: yellow">质的规定和量的规定</mark>两个方面
					- 价值的质的规定回答的是价值的实体是什么
					- 价值的量规定回答价值的大小由什么决定， 怎样决定
				- 决定商品价值量的不是生产商品的个别劳动时间，<mark style="background: transparent; color: yellow">而是社会必要劳动时间</mark>
					- 社会必要劳动时间时在现有的社会正常的生产条件下， 在社会平均的劳动熟练程度和劳动强度下制造某种使用价值所需要的劳动时间
			- 货币的产生和价值形式的发展
				- 商品价值形式的发展经历了四个阶段， 即简单的或偶然的价值形式，总的和扩大的价值形式，一般价值形式，货币形式
				- 货币的职能
					- <u>价值尺度，流通手段(最基本职能)</u>，贮藏手段，支付手段，世界货币等等。
		- 价值的规律及其作用
			- 价值规律和主要内容和客观要求是:<mark style="background: transparent; color: red"> 商品的价值量由生产商品的而社会必要劳动时间决定，商品交换以价值量为基础， 按照等价交换原则进行</mark>
			- 价值规律贯穿与商品经济的全部过程， 既支配社会生产，又支配社会流通。 
			- 价值的规律表现形式是: <mark style="background: transparent; color: lime">商品价格围绕商品的价值自发波动，由于供求关系的影响，商品价格围绕商品的价值自发波动，由于受到供求关系的影响，商品价格时而高于价值，时而低于价值 </mark>
			- 价值规律在市场配置资源过程中的作用
				- 自发调节生产资料和劳动力在社会各个生产部门之间的分配比例
				- 自发刺激社会生产力的发展
				- 自发调节社会收入的分配
		- 以私有制为基础的商品经济的基本矛盾
			- <mark style="background: transparent; color: lime">商品经济的基本矛盾是私人劳动和社会劳动的矛盾</mark>， 这一矛盾贯穿商品经济的发展始终，决定着商品经济的各种内在矛盾及其发展趋势
				- 私人劳动和社会劳动的矛盾<mark style="background: transparent; color: violet">决定着商品经济的本质(交换)以及发展过程</mark>
				- 私人劳动和社会劳动的矛盾<mark style="background: transparent; color: violet">是商品经济其他一切矛盾的基础</mark>
				- 私人劳动和社会劳动的矛盾<mark style="background: transparent; color: violet">决定着产品生产者的命运</mark>
			- 私人劳动和社会劳动的矛盾在资本主义制度下进一步发展为资本主义的基本矛盾，即生产社会化和生产资料资本主义私人占有的矛盾
		- 马克思劳动价值论
			- 通过对商品关系的深刻分析，马克思创立了<mark style="background: transparent; color: yellow">劳动价值论</mark>，主要包括<mark style="background: transparent; color: gray">商品的二因素和生产商品的劳动的二重性及其相互关系的理论，价值量的规定及其变化规律的理论</mark>

- <mark style="background: transparent; color: yellow">资本主义经济制度的本质及规律</mark>
	- 社会的演进和更替是生产力和生产关系，经济基础和上层建筑矛盾的必然结果
		- 萌芽
			- 封建社会末期，手工业得以较快发展，手工作坊的师徒关系转变为雇佣关系，成为资本主义生产关系萌芽的形式之一
			- 随着商人入侵手工业邻域，小生产者等便成为雇佣工人，商人或高利贷者成为资本家，这是资本主义生产关系萌芽的另一种形式
		- 资本原始累积
			- 其概念是以暴力手段是生产者和生产资料相分离，资本迅速集中于少数人手中
			- 主要是通过两个手段进行的，一是使用暴力手段剥夺农民的土地，二是使用暴力手段掠夺货币财富
			- 利用国家争取拿的力量进行残酷的殖民掠夺是资本原始累积的哟一个重要方式，如三角贸易，殖民统治等等，另外国债，课税制度和保护关税制度，加强对国内人民的剥削等等，都累积起了巨额的货币资本
		- 制度确立
			- 资产阶级政权统治建立和资本主义生产方式支配地位的形成标志着资本主义制度最终确立
	- 劳动力成为商品，货币转化为资本
		- $W-G-W$公式是商品流通公式， $G-W-G'$公式是资本流动公式，$G'$是增殖之后的货币，这个公式表明<mark style="background: transparent; color: orange">资本运动的一般目的是价值增殖</mark>，也称为“资本总公式”
		- <mark style="background: transparent; color: violet">劳动力的价值由生产，发展，维持，延续劳动力必需的生活必需品的价值决定</mark>
			- 1 维持劳动者本人生存必需的生活资料的价值 2 维持家属的生存必需的生活资料的价值 3 接受教育和训练支出的费用
			- 维持劳动者本人生存必需的生活资料的价值是劳动力价值的最低界限
		- 剩余价值
			- 由于劳动力商品的使用价值是价值的源泉，在消费过程中，可以创造新的价值，因此厚壁所有者购买到劳动力后可以得到一个增殖的价值即剩余价值$m$
	- 资本主义所有制
		- 资本主义所有制的含义
			- 经济意义上的所有制是指事实上生产资料归谁所有，支配和凭借支配获得剩余产品
			- 法律意义上的所有制是指生产资料占有关系的法律形态
			- 所有制决定所有权，所有制是所有权的基础，所有权是所有制的法律形态
			- <mark style="background: transparent; color: lime">资本主义所有制是生产资料归资本家所有的一种私有制形式，在资本主义所有制条件下，资本家拥有生产资料的所有权，劳动者和生产资料相分离</mark>
		- <mark style="background: transparent; color: red">资本主义所有制的本质</mark>
			- <mark style="background: transparent; color: lime">资本主义制度本质上是剥削制度</mark>
			- 资本家日通过购买工人的劳动力，将出卖了拉动力的劳动者和生产资料结合在一起进行生产并取得剩余价值
			- 资本家和劳动者的关系是资本和雇佣劳动的关系，由于在雇佣劳动中资本家凭借对生产资料的占有，<mark style="background: transparent; color: yellow">在等价交换原则的掩盖下，无偿占有雇佣工人创造的剩余价值，资本与雇佣劳动的关系由此具有了剥削与被剥削的对抗性质</mark>
		- 生产剩余价值是资本主义生产方式的绝对规律
			- <mark style="background: transparent; color: yellow">资本主义生产的直接目的和决定性动机，就是无休止地获取尽可能多的剩余价值</mark>
			- 剩余价值生产过程和资本不同部分在剩余价值生产中的作用
				- 资本主义的生产过程具有二重性
					- <mark style="background: transparent; color: lime">一方面是生产物质资料的劳动过程</mark>
						- 劳动过程包括劳动者的劳动，劳动对象和劳动资料，三个要素都为资本家所有
					- <mark style="background: transparent; color: lime">另一方面是生产剩余价值的过程</mark>
						- 剩余劳动无偿为资本家生产剩余价值，剩余价值体现了资本家和雇佣工人剥削与被剥削的关系
				- 资本可以划分为不变资本$c$和可变资本$v$
					- 资本主义的商品价值为$W= c+v+m$
					- $m' = \frac{m}{v}= \frac{剩余劳动}{必要劳动}$
			- 剩余价值生产的两种基本方法: 绝对剩余价值的生产和相对剩余价值的生产
				- 绝对剩余价值生产是延长工作时间或提高工作强度
				- 相对剩余价值是指在工作日长度不变的条件下，缩短必要劳动时间
			- 剩余价值的资本化，就是资本累积 
				- 其历史趋势是资本主义制度的必然灭亡和社会主义制度的必然胜利
		- 资本主义的基本矛盾和经济危机
			- <mark style="background: transparent; color: yellow">资本主义的基本矛盾是生产社会化和生产资料资本主义私人占有之间的矛盾</mark>
			- 生产过剩是资本主义经济危机的本质特征，其爆发的根本原因是资本主义的基本矛盾
			- 资本主义的经济危机具有周期性

- 资本主义政治制度和意识形态
	- 资本主义政治制度及其本质
		- 资本主义政治制度在资本主义经济基础上建立，反映了资本主义社会的经济关系，反映了政治上占据统治地位的资产阶级要求，同时资本主义制度作为上层建筑，反过来保护其经济基础， <mark style="background: transparent; color: yellow">本质上为资产阶级利益服务</mark>
		- 资本主义国家的职能和本质
			- <mark style="background: transparent; color: yellow">资本主义国家的职能以服务与资本主义制度和资产阶级利益为根本内容，包括对内实行政治统治社会管理，对外进行国际交往，维护国家安全利益两个基本方面</mark>
		- 资本主义的民主制度及其本质 
	- 资本主义意识形态的本质
		- 资本主义意识形态是资本主义社会的观念上层建筑，是作为资本主义的经济基础服务的，因而是为资本主义国家的政治上层建筑服务的
		- 资本主义意识形体态是资产阶级意识的集中体现


#### 6. 资本主义的发展及其规律

- 垄断资本主义的形成和发展（这一块多看课本）
	- 资本主义发展的两个阶段: 自由竞争资本主义和垄断资本主义
		- 自由竞争引起资本集中而引起垄断
	- 垄断资本的实质在于垄断利润， 有几下几个获取
		- 剥削本国无产阶级和其他劳动人民
		- 垄断高价和垄断低价来控制市场
		- 通过加强对其他国家的劳动人民的剥削
		- 通过资本主义国家政权进行再分配
	- 垄断价格不能增加和减少社会生产的价值总量
	- 垄断资本向世界范围的扩展主要有三种基本形式: 
		- 借贷资本输出
		- 生产资本输出
		- 商品资本输出
	- 资本主义发展到垄断资本主义，进而发展到帝国主义
	- 经济全球化及其影响：
- 资本主义为社会主义所替代的历史必然性
	- <mark style="background: transparent; color: yellow">资本主义的内在矛盾决定了资本主义必然被社会主义取代</mark>
	- <mark style="background: transparent; color: yellow">资本累积推动资本主义基本矛盾不断计划并最终否定资本主义自身</mark>
	- 国家垄断资本主义是资本社会化的更高形式
	- 资本主义社会存在资产阶级和无产阶级两大阶级的矛盾和斗争
