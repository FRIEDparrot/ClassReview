## 一、随机样本
我们将试验的全部可能观察值称为**总体**，而总体中的每一个观察值称为**个体**, 总体中包含的个体个数称为**总体的容量**。
首先每个个体是一个观察值， 而认为其是某一随机变量$X$的值, 即每个总体对应一个随机变量$X$.

由于各次观察都是在相同的条件下独立进行的，由于$X_1, X_2, \dots X_n$是相互独立的，可以称其为一个<mark style="background: transparent; color: red">简单随机样本</mark>。当n次观察完成之后，则

定义: 设$X$是具有分布函数$F$的随机变量，如果$X_{1}, X_{2}\dots  X_{n}$是具有同一分布函数$F$的，相互独立的随机变量， 则称$X_{1}, X_{2}, \dots X_{n}$为从分布函数$F$得到的容量为$n$的简单随机样本，简称<mark style="background: transparent; color: red">样本</mark>。其观察值$x_{1}, x_{2}  ,x_{3}\dots x_{n}$, 是随机变量的==观察值==， 称为<mark style="background: transparent; color: red">样本值</mark>。也称为$X$的$n$个独立的观察值。
因此，若有$X_1, X_2, \dots  X_n$是$F$的一个样本, 则其相互独立, 且分布函数均为$F$时, $(X_1, X_2, \dots X_n)$ 
$$F^{*} (x_{1} , x_{2}, \dots  x_{n}) = \prod_{i=1}^{n} F(x_{i})$$
若$X$具有概率密度$f$, 则$(X_1, X_2, \dots X_n)$的概率密度为:
$$f^{*} (x_{1} , x_{2}, \dots  x_{n}) = \prod_{i=1}^{n} f(x_{i})$$
## 二、直方图和箱线图
### (1) 直方图
直方图的每个小区间上高度为$\frac{f_{i}}{n}/ \Delta$, 其中$\Delta = \frac{\max - \min}{n}$ 
每个小矩形的面积为$\frac{f_i}{n}$, 面积 = 频率 $\rightarrow$ 概率

### (2) 箱线图 
<mark style="background: transparent; color: red">样本分位数的概念</mark>: 设有容量$n$的样本, 其观察值$x_1, x_2, \dots x_{n}$, 样本的$p$分位数记为$x_{p}$, 并且具有性质: (1) 至少有$np$个观察值小于或等于$x_{p}$; (2) 至少有$n(1-p)$个观察值大于或等于$x_p$, 则称$x_{p}$为**样本的$p$分位数**, 有:
$$x^{p}=  \begin{cases}
x_{([np] + 1)}   \qquad  np \text{ isn't integer}\\
\frac{1}{2} (x_{[np]} + x_{[np ] + 1}) \qquad  np \text{ is integer}
\end{cases}$$
样本$p$分位数的求法 : 
1. 若$np$不是整数, 则只有一个数据满足定义中的两点要求, 此时这一数据位于$[np] + 1$ 处, 
2. 若$np$是整数, 则可能有<u>多个数据满足定义中的两点要求</u>, 此时这个数据位于$\frac{1}{2}[x_{(np)} + x_{(np+1)}]$处

常用的几个分位数有:
1. $M(p = 0.5)$ 称为<u>样本中位数</u> 
2. 第一四分位数$Q_{1} = x_{0.25}$, 第三四分位数$Q_{3} = x_{0.75}$

我们称第一四分位数和第三四分位数$Q_{1}, Q_{3}$距离为<mark style="background: transparent; color: red">四分位数间距</mark>, 记为
$$Q_{3} - Q_{1} = IQR$$
且认为当数据Q<mark style="background: transparent; color: violet">有以下情况时认为是数据中的异常值</mark>:
$$Q < Q_{1} - 1.5IQR \qquad \text{or}\qquad  Q  > Q_{3} + 1.5 IQR$$

箱线图是基于以下5个数的图形: 即最小值Min, 第一四分位数$Q_{1}$, 中位数$M_{1}$, 第三四分位数$Q_{2}$, 最大值Max, 且绘制出的图形如下
![[Excalidraw/第六章 样本及抽样分布 2023-10-17 09.48.14|550]]
从箱线图可以看出<mark style="background: transparent; color: red">箱线图的如下性质</mark>: 
1. 中心位置: 中位数所在位置是数据集的中心(两侧个数相同)
2. 散布程度: 在图中，<mark style="background: transparent; color: red">每个区块(Min -Q1, Q1-M, M-Q2, Q2-Max)</mark> 的数据各占$\frac{1}{4}$, **某个区块较短时**表示**落在区间的点较为集中**, 长表示较为分散
3. 对称性: 当中位数位于箱子中间位置时, 数据分布较为对称，而如果数据分布较倾斜，可以看出数据的总体分布倾斜特性。
另外根据上述原则确定出异常值后, ==除去异常值==并使用 $*$ 表示异常值, 则得到<b><mark style="background: transparent; color: blue">修正箱线图</mark></b> 

## 三、抽样分布
### (1) 样本的常用统计量
统计量: 设$X_1, X_2, \dots  X_n$是总体$X$的一个样本, 而$g(X_{1}, X_{2}, \dots X_{n})$是$X_1, X_2, \dots  X_n$的函数, 若g中<mark style="background: transparent; color: red">不含未知参数</mark>，则称$g(X_1, X_2, \dots  X_n)$是一<mark style="background: transparent; color: red">统计量</mark>

此时定义常用的样本统计量如下:
样本平均值: 
$$\overline{X} = \frac{1}{n} \sum^{n}_{i=1} X_{i}$$
样本的方差
$$S^{2} = \frac{1}{n-1} \sum^{n}_{i=1} (X_{i} - \overline{X})^{2} = \frac{1}{n-1}\left(\sum^{n}_{i=1} X_{i}^{2} - n \overline{X}\right)^{2}$$
样本的标准差:
$$ S = \sqrt{S^{2}} = \sqrt{\frac{1}{n-1} \sum^{n}_{i=1} (X_{i} - \overline{X})^{2}}$$
样本的$k$阶原点矩:
$$A_{k} = \frac{1}{n}\sum^{n}_{i=1} X^{k}_{i}, \quad k = 1, 2\dots $$
样本的$k$阶中心矩:
$$B_{k} = \frac{1}{n} \sum^{n}_{i=1} (X_{i} - \overline{X})^{k}\quad  k= 2, 3\dots $$
需要说明，如果用总体中样本观察获取的值，则上述均使用小写字母表示，并记为<b><mark style="background: transparent; color: blue">样本的观察值</mark></b> 

对于样本的$k$阶矩$A_{k}$, 设$\mu_{k}$为[[📘ClassNotes/📐Mathmatics/🎣Probability Theory/第四章 随机变量的数字特征#一、随机变量的数学期望|总体的k阶矩]], 设总体的$k$阶矩$E(X^{k}) = \mu_{k}$存在, 则当$n \rightarrow \infty$时, $A_{k}\overset{P}{\rightarrow} \mu_{k}$, 这是因为$X_1, X_2,\dots$ 独立且与$X^{k}$同分布, 则
$$E(X^{k}_{1}) = E(X_{2}^{k}) = \dots E(X_{n}^{k}) = \mu_{k}$$
此时由[[📘ClassNotes/📐Mathmatics/🎣Probability Theory/第五章 大数定律以及中心极限定理#(2) 辛钦大数定律|辛钦大数定律]]可知, 
$$A_{k} = \frac{1}{n}\sum^{n}_{i=1} X^{k}_{i} \overset{P}{\rightarrow } \mu_{k}$$
并有(g为连续函数):
$$g(A_{1}, A_{2}, \dots A_{k}) \overset{P}{\rightarrow} g(\mu_{1}, \mu_{2} ,\dots \mu_{k})$$

### (2) 经验分布函数
对于总体分布函数$F(x)$, 设$X_1, X_2, \dots$是总体的一个样本, 并使用$S(x), -\infty < x < \infty$表示==其中不大于$x$的随机变量个数==, 则定义<mark style="background: transparent; color: red">经验分布函数</mark>$F_{n}(x)$为:
$$F_{n}(x) = \frac{1}{n} S(x)\qquad -\infty < x < +\infty$$
一般地, 若将$x_1, x_2, \dots$ **按照从小到大次序排列给出**, 则有:
$$F_{n}(x) = \begin{cases}
0  \qquad x < x_{1} \\
\frac{k}{n}  \space \space \quad  x_{k}\\
1 \qquad  x \geq  x_{n}
\end{cases}$$
对于此分布函数, Glivenko 证明了对于任意实数$x$, 当$n  \rightarrow \infty$时, $F_n(x)$以概率1一致收敛于分布函数$F(x)$, 即有:
$$P\left\{\lim_{n \rightarrow \infty} \sup_{-\infty < x < \infty}|F_{n}(x) - F(x)| = 0 \right\} =1$$
#### 1) $\chi^{2}$分布
我们设$X_1, X_2, \dots X_n$是<b><mark style="background: transparent; color: blue">来自总体N(0,1)的样本(即正态分布的样本)</mark></b>, 则称统计量
$$\chi^{2} = X_{1}^{2} + X_{2}^{2} +\dots + X_{n}^{2}$$
服从自由度为$n$的$\chi^{2}$分布, 记为$\chi^{2}\sim \chi^{2}(n)$
其中$\chi^{2}(n)$的概率密度为:
$$\Large f(y) = \begin{cases}
\frac{1}{2^{\frac{n}{2}} \Gamma \left(\frac{n}{2}\right)} y^{\frac{n}{2}-1} e^{-\frac{y}{2}} \qquad y > 0 \\
0 \qquad \text{others}
\end{cases}$$
![[attachments/Screenshot_20231017_142351_com.microsoft.skydrive.png|450]]
需要说明的是, $\chi^{2}$分布具有如下性质
1. 可加性: $\chi_{1}^{2}\sim \chi^{2}(n_{1})$, $\chi_{2}^{2} \sim \chi^{2}(n_{2})$, 则有:
$$\chi_{1}^{2} + \chi_{2}^{2}\sim \chi^{2}(n_{1} + n_{2})$$
2. 数学期和方差
若有$\chi^{2}\sim\chi^{2}(n)$, 则有:
$$E(\chi^{2}) = n\qquad  D(\chi^{2}) = 2n$$

$\chi^{2}$分布的分位点: 对于正数$\alpha$, 且$0<\alpha<1$, 则称满足条件
$$P\left\{\chi^{2} >\chi^{2}_{\alpha}(n) \right\} = \int_{\chi^{2}_{\alpha}(n)}^{\infty}f(y)dy = \alpha $$

#### 2) t分布
设$X\sim N(0,1), Y\sim \chi^{2}(n)$, 且X,Y相互独立, 则称随机变量
$$t = \frac{X}{\sqrt{Y/n}}$$
服从于**自由度为$n$的$t$分布**, 概率密度函数
$$h(t) = \frac{\Gamma[\frac{n+1}{2}]}{\sqrt{\pi n} \Gamma(\frac{n}{2})} \left(1 + \frac{t^{2}}{ n}\right)^{-\frac{n+1}{2}}\qquad  -\infty < x < \infty$$
![[attachments/Screenshot_20231017_145726_com.microsoft.skydrive.png|500]]
对于给定的$\alpha$, 0 < $\alpha$ < 1, 称满足条件
$$P\left\{t> t_{\alpha}(n)\right\} = \int_{t_{\alpha}(n)}^{\infty} h(t) dt = \alpha$$
为$t_{\alpha}(n)$的上$\alpha$分位点, $n$足够大时,参考[[📘ClassNotes/📐Mathmatics/🎣Probability Theory/第二章 随机变量及其分布#3. 正态分布|正态分布]], 则通过[[Excalidraw/第六章 样本及抽样分布 2023-10-18 08.57.36|简单推导]]可以得到
$$\lim_{n \rightarrow  \infty} h(t) = \frac{1}{\sqrt{2\pi}} e^{-\frac{t^{2}}{2}}$$
即$n\rightarrow \infty$时, t分布变为参数为$(0,1)$的正态分布。
t分布有性质:
$$t_{1- \alpha}(n) = -t_{\alpha}(n)$$
![[Excalidraw/第六章 样本及抽样分布 2023-10-18 08.48.26]]
对于如上的$\alpha$分位点可以表示为:
$$P(t > t_{\alpha}(n)) = \int_{t_{\alpha}(n)}^{\infty} h(t) dt = \alpha$$
#### 3) F分布
设$U\sim \chi^{2}(n_{1}), V\sim \chi^{2}(n_{2})$且U,V相互独立, 则称随机变量
$$F = \frac{U/n_{1}}{V/n_{2}}$$
服从于自由度为$(n_1, n_2)$的$F$分布, 记为$F\sim F(n_1, n_2)$
分布概率密度为
$$\Large \psi(y) = \begin{cases}
\frac{\Gamma\left[\frac{n_{1} + n_{2}}{2}\right]\left(\frac{n_{1}}{n_{2}}\right)^{\frac{n_{1}}{2}} y^{\frac{n_{1}}{2} -1}}{\Gamma\left(\frac{n_{1}}{2}\right)\Gamma\left(\frac{n_{2}}{2}\right) \left[1 +\frac{n_{1}}{n_{2}} y\right]^{\frac{n_{1}  + n_{2}}{2}}} \qquad y> 0 \\
0\qquad  \text{others}
\end{cases}$$
![[Excalidraw/第六章 样本及抽样分布 2023-10-18 09.56.07]]
显然有**结论**: 若有$F\sim F(n_1, n_2)$, 则
$$\frac{1}{F} \sim F(n_{2}, n_{1})$$
F分布的分位点: 称满足条件
$$P\left\{F > F_{\alpha}(n_{1}, n_{2}) \right\} =  \int_{F_{\alpha}(n_{1}, n_{2})}^{\infty} \psi(y) dy = \alpha$$
的点为$F(n_1,n_2)$的上$\alpha$分位点, 且有性质:
$$F_{1- \alpha} (n_{1},n_{2})  = \frac{1}{F_{\alpha}(n_{2}, n_{1})}$$
## 四、 正态总体样本均值与样本方差的分布
设总体的均值为$\mu$, 方差$\sigma^2$,而$X_1, X_2, \dots X_n$是来自$X$的一个样本, $S^{2}$分别是均值与方差, 则有:
$$E(\overline{X}) = \mu\qquad D(\overline{X}) = \frac{\sigma^{2}}{n}$$
而
$$E(S^{2}) = \sigma^{2}$$
进而有$\overline{X}$也服从正态分布, 从而有: 
**定理**: 设$X_1, X_2, \dots X_n$是来自总体$X$的一个样本, 则:
$$\overline{X} \sim N\left(\mu, \frac{\sigma^{2}}{n}\right)$$
