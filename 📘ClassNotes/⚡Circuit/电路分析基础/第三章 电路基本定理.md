## 一、叠加定理和齐次定理
### (1) 叠加定理
叠加定理是指**在电路中， 所有==独立电源==同时作用时， 在每一个支路中产生的响应电流或者电压，等于==各独立电源单独作用==时在该支路中产生的响应电流或者电压的代数和**。

这一结论称为叠加性。说明了线性电路中独立电源的独立性。

需要说明的是: 
1. **独立的电源进行单独作用**时， 其余的电源中: <mark style="background: transparent; color: red">电压源使用短路代替， 而电流源使用开路代替</mark> 
2. 最后叠加时， 必须注意<mark style="background: transparent; color: red">各个响应(电压，电流)的分量是代数和</mark>。注意在最终叠加时的正负号问题。
3. 电路含有受控源时， 则==在任何情况下，受控源都应当被保留==。
4. 叠加定理<u>不能用于计算电路的功率</u>，由于功率是电压流或者电压的二次方函数。

### (2) 齐次定理
齐次定理是指**在线性电路中，当全部的激励同时增大$k$倍时，对应的响应也增加$k$倍**
齐次定理反映了电路的响应和激励成正比。需要注意的是， 齐次定理与叠加定理是两个完全不同的定理， 相互独立。

注意: 全部的激励同时增大

## 二、替代定理
替代定理指电路中的任意$k$条支路的**电压和电流已知时**， 则这条支路可以使用一个电压等于$u_k$的<mark style="background: transparent; color: red">理想电压源</mark>或者电流为$i_k$的<mark style="background: transparent; color: red">理想电流源</mark>进行代替。

需要注意的是，替代与等效概念不同。替代不代表等效。

## 三、等效电源定理
### (1) 单口网络的分析与计算 
单口网络: 将具有<mark style="background: transparent; color: red">两个引出端钮且连上的电流为同一电流的部分电路</mark>称为单口电路或者单口网络。单口网络可以分有源单口网络可无源单口网络。
![[Excalidraw/第三章 电路基本定理 2023-09-08 14.26.32|800]]
我们对于无源单口网络， 可以使用等效电阻$R_{in}$表示， 因而有$R_{in} = \frac{u}{i}$为**输入电阻**
`````ad-note
title: 有源网络输入电阻
collapse: open
对于有源网络的输入电阻， 先将网络的独立电压源和电流源置零，
即**电压源换成导线**，而**电流源换成开路**
`````
定义无源单口网络的输入电导: 
$$G_{in} = \frac{1}{R_{in}} = \frac{i}{u}$$
另外需要考虑单口网络中含有受控源的情况, 如上图中的2,3所示: 先计算出对应的<u>电压下的两端电流值</u>， 也可以<u>加一个电流源并算出电压值</u>。从而确定对应的等效电阻。

1. 仅含电阻时， 可以直接利用串并联和$\Delta-Y$变换等等求解电阻
2. **含有受控源**时， 可以利用电压电流法求解输入电阻

### (2) 等效电压源定理(戴维南定理)
等效电压源定理: 对于<mark style="background: transparent; color: red">任何一个线性的有源单口网络</mark>，对于外电路来说明都可以使用一个电压源$u_{oc}$ 和一个电阻$R_{o}$的**串联组合**来等效代替。
戴维南定理的图示如图:
![[Excalidraw/第三章 电路基本定理 2023-09-08 16.25.09|350]]
其中对于等效出电压的大小的计算，显然有公式: 
$$\Large \boxed {u = u_{oc} - R_{0} i}\tag{4.8}$$
其中，$R_0$ 数值上等于网络$N$除去所有源之后的电阻, 而$u_{oc}$为线性含源网络的<mark style="background: transparent; color: red">开路电压</mark>

`````ad-note
title: 戴维南定理的计算方法(重点)
collapse: open
1. 当有源时，可以求解**开路电压 + 短路电流**$U_{oc}/I_{sc}$方法
$$\Large\boxed{U= U_{oc} \qquad  R = \frac{U_{oc}}{I_{sc}}}$$
2. 如果没有受控源， 可以使用串并联方法或者使用<mark style="background: transparent; color: red">开路电压 + 等效电阻</mark>方法
3. 当含有受控源时， 既可以使用$U_{oc}/I_{sc}$得到， 也可以使用外加电压或者电流源方法，列出系统的方程进行求解
`````

### (3) 等效电流源定理(诺顿定理)
任何一个线性的有源端口网络， 对于外电路，可以使用一个电流源和一个电阻$R_0$的**并联组合**进行代替。(也是相当于电源)
其中， **等效电流源定理**中，电流源大小为$I_{sc}$, 而电阻大小可以利用串并联等求解。
`````ad-note
title: 诺顿定理的求解方法
collapse: open
对于短路电流比较好求解的电路(例如双Y型), 可以利用**短路电流和等效电阻**进行求解。

`````
诺顿定理和戴维南定理的两个基本关系为:
$$\Large \boxed{i_{sc} = \frac{u_{oc}}{R_{o}}}\tag{4.10}$$
其中$u_{oc}$为开路电压，$R_o$为网络$N$除去所有源后的**除源电阻**。因此也称==电源变换==为<mark style="background: transparent; color: red">戴维南-诺顿变换</mark> 

`````ad-caution 
title: 电路特性与戴维南-诺顿电路
collapse: open
对于任何一个线性网络, 对于外部的电压-电流特性可以表征为:
$$U = A - IB$$
其中$A$为等效电压源的$U_{oc}$, $B$为等效电阻的大小

上式也可以化为
$$I = C - \frac{U}{D}$$
其中$C$为诺顿定理中电流源的大小, $D$为诺顿定理的等效电阻

另外
如果一个端口网络的等效电阻$R_{eq} =  0$, 则该端口仅有戴维南等效电路，没有诺顿等效电路
如果一个端口网络的等效电阻$R_{eq} = \infty$, 则该端口无戴维南等效电路，只有诺顿等效电路
`````

## 四、最大功率传输定理 
对于一个如图所示含源线性的二端网络，则可以利用戴维南定理和诺顿定理推导出下方的模型: 
![[Excalidraw/第三章 电路基本定理 2023-09-08 17.09.24|600]]
通过[[📘ClassNotes/⚡Circuit/电路分析基础/补充部分/最大功率传输定理推导.pdf|最大功率传输定理推导.pdf]]得到左侧和右侧的最大功率分别为:
$$\boxed{P_{1, \max} = \frac{u_{oc}^{2}}{4R_{0}} \qquad P_{2, \max} =\frac{1}{4} i_{sc}^{2} R_{o}}$$
两者==均在$R = R_0$的状态下能够取得功率的最大值==。此时效率很低$\eta_{p  =\max} = \frac{P_{\max}}{P} = \frac{1}{2}$

对于一个线性网络的最大功率， 我们可以通过戴维南和诺顿变换以及最大功率传输定理得到。最大功率传输定理回用于一端口的电路已经给定， 而电阻可调的情况下时给常方便。

需要说明的是, <b><mark style="background: transparent; color: blue">效率和功率不同</mark></b>， 效率是$R_L$越大越好, 而功率是不同的。

## 五、特勒根定理
特勒根定理1(功率守恒定理): 对于一个具有n个结点和b条支路的<mark style="background: transparent; color: red">集总电路</mark>, 在支路电流和电压取关联的参考方向的情况下, 满足:
$$\Large\boxed{\sum^{b}_{i=1} u_{k} i_{k} = 0}$$
即任何一个电路的全部支路吸收的功率等于发出的功率。(或者吸收功率等于发出功率)
![[Excalidraw/第三章 电路基本定理 2023-10-29 17.05.06|500]]
特勒根定理2(拟功率守恒定理): 对于具有相同拓扑结构的集总电路, 要求两个均具有n个节点和$b$条支路的集总电路, 有相同的图， 但是由内容不同的支路构成，在支路电流和电压取关联参考方向的情况下，有: 
$$\boxed{\sum^{b}_{k=1} u_{k} \hat{i}_{k} = 0 \qquad  \sum^{b}_{k=1} \hat{u}_{k}i_{k} = 0}$$
上述**两个特勒根定律证明**容易使用KCL和节点电压方法进行证明并推广到每一个电路。
`````ad-cite 
title: 说明
collapse: open
特勒根定理和KCL, KVL定律均适用于集总电路， 都是对拓扑结构的约束。
实际使用时， 特勒根定理2在实际使用中更为重要。往往对于$P$(无源线性网络)和$A$(含源线性网络)
`````

`````ad-caution 
title: 注意事项
collapse: open
特勒根定理要求参考方向必须一致, 如果是第二类特勒根定理， 则参考方向要看$u_k$和$\hat{I}_{k}$的方向是否一致。分别要看两个图的电压。另外， **从电源流出的电流一定是非关联参考方向的, 应当注意正负号的添加。但第二类特勒根定理要看$\hat{I}_k$与$u_{k}$的方向是否实际关联**
`````

## 六、互易定理
对于一个<mark style="background: transparent; color: red">线性无任何电源</mark>的受控电路(具有互异性的网络)， 在<mark style="background: transparent; color: red">只有一个激励源的情况</mark>下，==激励与响应互换位置， 响应与激励比值不变==。此结论说明了线性无任何电源的电路传输信号的双向性或者可逆性。互易定理是利用特勒根定理进行证明的。

互易定理有<b><mark style="background: transparent; color: blue">三种基本形式</mark></b>， 如下图所示:
![[Excalidraw/第三章 电路基本定理 2023-09-08 19.13.33|1000]]
第一种形式: 若在$1-1'$端加激励源$u_{s1}$, 则2-2端的短路电流$\hat{i}_1$为响应，而若在2-2'端施加激励$u_{s2}$时, 并以$1-1'$中的短路电流$i_2$为响应，则当$u_{s2} = u_{s1}$时，有$\hat{i}_1 = i_2$, 即<mark style="background: transparent; color: red">左端电压激励时，右端测量电流的大小</mark>和<mark style="background: transparent; color: red">右端作用相同电压时左端的响应电流大小相同</mark>, 或者有:
$$\frac{i_{2}}{u_{s1}} = \frac{i_{1}}{u_{s2}}$$

第二种形式: 若在$1-1’$端接电流源$i_s$，则$2-2'$端开路电压$u_2$为响应, 若把激励$i_s$移到$2-2’$端时，此时有$1- 1'$端的开路电压为$\hat{u}_1 = u_2$. 即<mark style="background: transparent; color: red">左端电流激励时，右端测量电压的大小</mark>和<mark style="background: transparent; color: red">右端作用相同电流时左端的响应电压大小相同</mark>
$$\frac{u_{2}}{i_{s1}} = \frac{u_{1}}{i_{s2}}$$

第三种形式: 若在$1-1'$端接电流源$i_s$，得到的$2-2'$端的短路电流为$i_2$为响应， 若在$2-2'$端接电压源$u_s$作为激励, 且在数值上$u_s = i_s$, 则图中两个开路电压为$\hat{u}_1 = i_2$，即<mark style="background: transparent; color: red">左端电流激励时，在右端测量出电流响应大小</mark>, 而<mark style="background: transparent; color: red">右端作用与原先左侧电流数值上相同的电压时左端的响应电流大小与原先右侧的电流响应相同</mark>
$$\frac{i_{2}}{i_{s1}} = \frac{u_{1}}{u_{s2}}$$
`````ad-caution 
title: 互易定理的注意要点
collapse: open
1. 互易前后应当保持网络的拓扑结构不变， 仅有理想电源搬移
2. 互易前后的端口处的激励和响应极性应当保持一致，否则等式要加负号(考虑非关联性)
3. 互易定理仅适用于线性电阻网络在单一电源的激励下的 两个支路电压电流的关系
4. 对于**含有受控源的网络, 互易定理一般不成立**
`````

互易定理在<u>分析网络的传输特性以及接受天线的方向性等方面具有广泛应用</u>。满足互易定理的网络称为互易网络。

## 七、对偶原理
1. R与G对偶关系, u,i 为对偶关系, C,L 为对偶关系
$$\begin{cases}
R = \sum^{n}_{k=1} R_{k}  \\
i = \frac{u}{R}  \\

\end{cases}$$
2. 串联与并联对偶
3. 电压源与电流源对偶
4. 结点和回路对偶
5. KVL, KCL对偶 
6. 星型与三角形对偶
7. 戴维南，诺顿原理对偶

`````ad-note
title: 说明
collapse: open
对偶的电路之间，如果研究对偶的结构，得到相同的结构
对偶的电路之间，如果研究相似的量，得到对偶的结构
往往我们可以找一个与已知电流的对偶电路，可以得到基本相同的定理
`````
