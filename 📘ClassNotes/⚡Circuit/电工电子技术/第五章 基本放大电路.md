### (2)放大电路的静态分析
#### 1. 方程法
对于一般的放大电路, <mark style="background: transparent; color: red">需要静态分析的参数</mark>为$I_C, I_{B}$和$U_{CE}$三个参数。
静态分析的方法**首先是画出直流通路(即电容视为开路)**, 
![[Excalidraw/第五章 基本放大电路 2023-11-14 09.22.30|600]]
首先，在一般的放大电路中, $I_{B}$都很小, 而$U_{BE}$往往是给出的, 因此我们可以先计算$I_B$, 有
$$I_{B} = \frac{U_{CC} - U_{BE}}{R_{B}}\approx \frac{U_{CC}}{R_{B}}$$
$$I_{C}  = \beta I_{B}\qquad I_{E}= (1 + \beta) I_{B}$$
`````ad-caution
title: 说明
collapse: open
在上述的通路中, 如果$U_{BE}$未给出， 可以忽略， 也可以按照硅管的0.6V进行计算
`````
此时, 利用右侧通路, 有关系:
$$ U_{CC} = {U_{CE} + I_{C}  R_{C}}$$
得到了放大电路静态分析的全部参数。
#### 2. 图解法与静态工作点的确定
根据上面的推导， 显然有:
$$I_{C} =  \frac{U_{CC}- U_{CE}}{R_{C}}$$
显然, 实际的$U_{CE}$必然是$I_{C}-U_{CE}$曲线与$U_{CC} - I_{C}R_{C}$的交点(如上图所示), 称为<b><mark style="background: transparent; color: blue">静态工作点</mark></b>

`````ad-summary
频率过高或者过低时放大倍数改变的现象与三极管电容效应和放大电路中的电容有关

对于共射极放大电路，有三个特点:
- 放大倍数高
- 输入电阻低
`````

首先, 我们绘制出交流通路, 由于此时直流电源相当于不作用，==此时电容短路并将$U_{CC}$接地(和E接到同一处), 得到交流通路==(图2), 然后将等效为一个电阻$r_{be}$和一个受控电流源
![[Excalidraw/第五章 基本放大电路 2023-10-08 12.22.03|1000]]
其中小功率晶体管的$r_{be}$ 可以利用<mark style="background: transparent; color: red">输入电阻近似公式</mark>计算(其中, $r_{b}$为三极管的基区的等效电阻, 小功率管的$r_b$约为200$\Omega$): 
$$\Large\boxed{r_{be} \approx r_{b} + (\beta +  1) \frac{26mV}{I_{E}}}$$
则由公式$I_{C }= \beta I_{B}$, <mark style="background: transparent; color: red">需要注意由于输出电流方向, 实际输出电压与输入电压相反</mark>, 则:
$$I_{b} = \frac{U_{i}}{r_{be}} \qquad U_{o}= -\beta I_{b}(R_{c}  \parallel R_{L})$$
得到<mark style="background: transparent; color: red">电压放大倍数</mark>$A_{u}$为
$$\Large\boxed{A_{u}= \frac{U_{o}}{U_{i}} = -\beta \frac{R_{c} \parallel R_{L}}{r_{be}}}$$
需要说明的是, 如果计算输出电压与电源电压的比值, 可以使用下式计算:
$$E_{s} \frac{R_{in}}{R_{s} + R_{in}} = U_{in} \qquad  \frac{U_{o}}{E_{s}} =\frac{R_{in}}{R_{in} + R_{s}} A_{u}$$
同样地, **输入电阻和输出电阻使用戴维南定理计算, 有**(直接为并联):
$$R_{in}= \frac{U_{i}}{I_{i}}= R_{B} \parallel r_{be}$$
输出电阻计算方法需要**使用外加电压方法的戴维南定理**, **首先移除电路内部所有理想电源**, 然后去除$R_{L}$并加上电压$U$, (左侧无电流时)显然本题得到的电阻为$R_{C}$:
$$R_{o} = R_{C}$$

`````ad-note
2. 除去内部电源
3. 在负载处加上电压$U$求解电流
`````

在动态放大电路的图解分析中，有:
$$U_{CE} = U_{CC} - i_{C} R_{C}$$

#### 3. 放大器产生的失真现象
如果实验选取的**静态工作点**过高 ($Q = Q_{H}$)或者静态工作点过低($Q = Q_{L}$), 此时电压可能无法继续增加，导致半周期内的部分成分失去, 分别称为$Q =Q_{H}$**削顶失真**(饱和失真)和$Q = Q_{L}$**削底失真**(截止失真), 因此为了保证放大电路有合适的$Q$点, 可以改变$R_{B}$来调节$I_{B}$大小。

## 二、共集电极放大电路
### (1) 静态分析和动态分析
**共集电极放大电路**也称为**射极输出器**, 基本结构如图1, 由于**对于交流信号而言, 其输出和输入的公共端是集电极(均接地), 称为共集电极电路**。
![[Excalidraw/第五章 基本放大电路 2023-11-14 15.20.45|650]]
**静态分析**参数计算(其中$U_{BE}$给出或者按照0.6V或者忽略):
$$I_{B}R_{B} + (1 + \beta) I_{B} R_{E}= U_{CC} - U_{BE}$$
$$I_{C}= \beta I_{B}, I_{E} = (\beta + 1) I_{B}, U_{CE }= U_{CC} - (1 + \beta) I_{B}$$
**动态分析**:
1. 放大倍数$A_u$(<mark style="background: transparent; color: violet">注意: 放大倍数有可能是负的, 表明相位角差</mark>): 
$$i_{B}r_{be} + (1 + \beta ) i_{B} R_{E} = U_{i}\qquad U_{o} = (1 + \beta) i_{B} R_{e}\qquad  A_{u} = \frac{U_{o}}{U_{i}} = \frac{(1 + \beta) R_{E}}{r_{be} + (1 + \beta)R_{E}}$$
<mark style="background: transparent; color: red">共集电极放大电路没有电压放大作用， 但是有电流放大和功率放大作用</mark> 
`````ad-caution 
title: 说明
collapse: open
其中, 如果不考虑两个电路的干扰， 则相当于将$R_{E}$乘系数(1 + $\beta$)之后分别并联在左右边， 比较简单的放大电路可以如此分析。
对于含有三极管的部分， 可以用网孔法做类似的题目
`````
2. 输入电阻 
$$r_{in} = \frac{U_{in}}{I_{in}} = R_{B} + r_{be} \parallel (1 + \beta) R_{e}$$
3. 输出电阻
对于共集电极放大电路而言，输出电阻的求解较为麻烦, 需要将$u_{o}$部分**除去负载并加上电源**进行求解:
![[Excalidraw/第五章 基本放大电路 2023-11-14 19.15.37|800]]
得到结果:
$$R_{eq}= \frac{(R_{be} + R_{s}') R_{e}}{R_{be} + R_{s}' + (1 + \beta)  R_{e}}\qquad R_{s}' = R_{s} || R_{B}$$
一般情况下, $\beta$ 远大于1, 则有:
$$R_{eq} \approx \frac{R_{be} + R_{s}'}{\beta }$$
### (2) 共集电极放大电路的应用
共集电极放大电路特点: <mark style="background: transparent; color: red">输入电阻大， 输出电阻小</mark>
可以用于做**输入级，输出级和中间隔离级**
1. 共集电极放大器常常作为多级放大电路的输入级，此时<mark style="background: transparent; color: red">输入级采用射极输出器，可以使信号源的内阻上有比较小的压降，得到较高的输入电压</mark>， 同时减小信号源提供的信号电流， 从而减轻信号源的负担。
2. 由于输出电阻低，所以负载电流变动较大时，输出的电压变化小， <mark style="background: transparent; color: red">带负载的能力强</mark>
3. 将射极输出器<mark style="background: transparent; color: red">接在两级共射放大电路之间</mark>作为隔离极, <mark style="background: transparent; color: lime">利用输入电阻高的特点提高前一级的电压放大倍数， 并利用输出电阻低的特性减小后以及的信号源的内阻</mark>, 从而起到隔离作用。

## 三、分压式偏置电路
### (1) 稳定工作点的原理
对于一般的放大电路，由于二极管的特性随温度的改变而改变, 因此温度升高, 一般放大电路的特性也会发生改变， 可以通过在直流通路上加入电阻$R_{E}$和$R_{B2}$进行解决

温度升高时, 相同$U_{be}$状态下的$I_C$升高, 导致三极管的特性改变。
![[Excalidraw/第五章 基本放大电路 2023-11-14 14.38.21|750]]
<b><mark style="background: transparent; color: blue">解决温度改变对工作点影响的原理</mark></b>: 
`````ad-cite
title: 一般条件
collapse: open
首先，上图中基极电流可忽略，有$I_{B} \approx I_{B2}$,且有条件$V_{B} >> u_{BE}$
`````
此时B点电势$V_{B} = I_{B} R_{B2} = \text{const}$, 又有$I_{C} \approx I_{E}$, 而E点电势为$V_{E} = V_{B} - U_{BE}$,即有$U_{BE} = V_{B} - V_{E}$ 此时, 由于$I_{C}$升高, $V_{E} \approx I_{C} R_{E}$增大, **则$U_{BE}$减小从而起到调节效果**, $I_{C}$减小，使得Q点基本稳定。

### (2) 静态和动态分析
静态分析时，计算$I_{B}, I_{C}$和$U_{CE}$: 
$$V_{B} = \frac{R_{B2}}{R_{B1} + R_{B2}} U_{CC}$$
$$I_{E} = \frac{V_{B }- U_{BE}}{R_{E}} \approx I_{C} = \beta I_{B}\qquad U_{CE} = U_{CC} - I_{C} R_{C} - I_{E} R_{E}$$
动态分析时, 交流通路基本相同, 则有:
$$I_{B} =\frac{U_{i}}{r_{be}} \quad  U_{o} = -\beta I_{B} (R_{C} \parallel R_{L})\rightarrow \quad A_{u} =  \frac{U_{o}}{U_{i}} = - \frac{\beta (R_{C} \parallel R_{L})}{r_{be}}$$
输入, 输出电阻: 
$$R_{in} = R_{B1} \parallel R_{B2} \parallel r_{be}\qquad  R_{o} = R_{c}$$

## 六、差分放大电路 
### (1) 静态分析
差分放大电路不仅可以放大缓变直流和交流信号, 也可以**有效抑制零点漂移**。往往用于多级直接耦合放大电路的前置级和集成运放的输入级。基本结构为下图 : 
![[Excalidraw/第五章 基本放大电路 2023-11-27 09.46.48|700]]
对于静态直流分析电路, 我们取$u_i = u_o = 0$, 则两管的集电极电流, 集电极电位均相等, 显然有:
$$I_{C1} = I_{C2} \qquad V_{C1} = V_{C2}$$
此时, 若环境的温度发生改变, 根据[[📘ClassNotes/⚡Circuit/电工电子技术/第四章 半导体器件#(4) 三极管的特性曲线和主要参数|三极管的特性曲线]], 在$U_{BE}$相同时, 温度升高, 两端$I_{C}$增加, 显然当$I_{C1}, I_{C2}$同时增加$\Delta I$时, 输出仍然为0。
$$V_{C1} = V_{C2} = V_{CC} - I_{C} R_{C} \qquad  u_{o }= V_{C_{1}} - V_{C2} = 0$$
输出端有关系: 
$$\boxed{U_{BE} = V_{B} - V_{E}}= -I_{B} R_{B} - ( -U_{EE} +2 I_{E}R_{E})$$
从而整理得到:
$$U_{EE} = U_{BE} + I_{B} R_{B} +2 I_{E} R_{E}$$
工程上往往有$U_{BE}+I_{B}R_{B} << 2I_{E}R_{E}$, 得到:
$$\Large\boxed{I_{C} \approx I_{E} = \frac{U_{EE}}{2 R_{E}}\qquad  I_{B}  = \frac{1}{\beta} I_{C}}$$
$$\Large \boxed{U_{CE} ={\small U_{CC} - I_{C} R_{C} - 2 I_{E} R_{E} + U_{EE}} = U_{CC} - I_{C} R_{C} }$$
`````ad-check
title: 说明
collapse: open
在差分放大电路中, 电阻$R_{p}$起调零作用, 也称为静态调零电位器。而$R_{E}$用于限制漂移的范围, 从而进一步减小零漂, 而$-U_{EE}$用于抵消$R_E$上的直流压降, 获取合适的静态工作点。
`````

### (2) 动态分析 
动态分析时, 根据不同的输入, 放大电路的等效电路不同, 有差模输入和共模输入两种输入方式
**差分放大电路对差模输入信号有放大作用，而对共模输入信号有抵消作用**。
显然共模输入有$u_o = 0$, 且任意输入$u_{i1}, u_{i2}$时, 可取差模和共模分解:
$$u_{c} = \frac{u_{i1} - u_{i2}}{2}\qquad  u_{d} = \frac{u_{i1} + u_{i2}}{2}$$
#### 1. 单管差模输入的动态分析
`````ad-note
首先我们说明**差模输入**和**单管差模输入**的概念: 
1. 差模输入时, $u_{i1} = -u_{i2}$, 如果$u_{i1}$引起$V_{C1}$变化$\Delta V_{C1}$, 则$u_{o} = 2\Delta V_{C1}$
2. 单管差模输入时, 我们可以取$u_{i2} = 0$，仅从一侧输入即可。
`````
在上方我们已经说明了$U_{EE} \approx 2I_{E} R_{E}$, 因此我们视下方为短路, 而**在单管差模信号输入时,  另一口输入**$u_{i2} = 0$, 可以视为短路, 从而有:
$$U_{BE} = V_{B} - V_{E} = I_{B} R _{B} \qquad  I_{B} = \frac{U_{BE}}{R_{B}} \approx 0, \quad I_{C2} \approx 0$$
因此$R_{C2}$可以视为短路, 仅有$R_{C1}$接在$u_o$的正负两端, 从而可以画出**图2的等效电路**。
![[Excalidraw/第五章 基本放大电路 2023-11-27 21.51.53|1100]]
将图2使用微变等效电路图3分析, 则有(其中$A_{d1}$为1侧单管输入放大倍数):
$$i_{B}= \frac{u_{i1}}{R_{B} + r_{be}}, \qquad u_{o} = -\beta u_{i1} \frac{R_{c}}{R_{B} + r_{be}}\qquad A_{d1} =  A_{d2}= A_{d}= - \frac{\beta R_{c}}{R_{B} + r_{be}}$$
从而**考虑负载电阻**(由于差模信号输入时, $R_{L}$可以认为是中点接地, 即中点与地连接)放大倍数变为: 
$$\boxed{A_{d} = - \frac{\beta (R_{C} \parallel \frac{1}{2}R_{L})}{R_{B} + r_{be}}}$$
为**差模信号放大倍数**, 其中系数1/2可以通过下图更容易理解:

对于差模输出, 输入电阻为
$$r_{i}  = 2 (R_{B} + r_{be})$$
#### 2. 共模抑制比与输出方式
理想情况下, **共模信号放大倍数**$A_{c} = 0$, 但实际中是不等于0的, 取
$$K_{CMRR}  =\left|\frac{A_{d}}{A_{c}} \right|$$
为**共模抑制比**, 如果取$20\lg \left|\frac{A_{d}}{A_{c}} \right|$时, 单位为dB, 实际差分放大电路的$K_{CMRR} \approx 10^{5} = 100dB$

差模电压放大倍数**和输出方式有关**, 而**与输入方式无关**。双端输出时, 差模电压放大倍数等于每一边的单管放大倍数的$A_d$, 而单端输出(另一端接地)时, 差模电压放大倍数为每一边的$A_d$的一半。需要注意此时的相位关系:
$$A_{d}= - \frac{1}{2} \frac{\beta R_{C}}{R_{B} + r_{be}}(反相输出)\qquad A_{d} = \frac{1}{2} \frac{\beta R_{C}}{R_{B} + r_{be}}(同相输出)$$
## 七、功率放大电路
### (1) 功放电路的基本工作状态
`````ad-abstract
title: 功率放大电路的基本要求
collapse: open
1. 较大的输出功率
$$P_{om} = \frac{U_{om}}{\sqrt{2}} \cdot \frac{I_{om}}{\sqrt{2}} = \frac{U_{om} I_{om}}{2}$$
实际工作中, 往往让晶体管工作在极限状态, 而不超过晶体管的极限参数($P_{CM}, I_{CM}, U_{(BR)CEO}$)
2. 较高的效率 
我们取直流电源提供的功率为$P_E$, 而输出功率为$P_{om}$, 则效率计算为:
$$\eta = \frac{P_{om}}{P_{E}} \times 100 \%$$
3. 非线性失真小
由于在工作过程中, 晶体管超出线性范围, 即使不发生饱和失真和截止失真， 也会有**非线性失真**存在
`````
在实际的应用中，放大电路不仅需要较大的输出电压, 还需要较大的输出功率用于推动执行机构工作。
功率放大电路有**甲类, 乙类和甲乙类工作状态**, <u>甲类工作状态将静态工作点设置在交流负载线的中点</u>, <u>乙类的静态工作点接近于截止区</u>, 而甲乙类略微上移。
![[Excalidraw/第五章 基本放大电路 2023-11-28 09.04.17|1000]]
甲类工作状态可以无波形失真地实现功率放大，但是**电源提供的功率主要损耗在晶体管的集电极, 理想效率也只有50%**(几乎不采用)
而乙类和甲乙类工作状态会产生较大的失真, 但电源效率很高。在实际中，为了解决高效率和非线性失真的矛盾， 常常采用互补对称功率放大电路, 可以**在提高效率的同时实现减小波形的失真**。
### (2) 互补对称功放电路
#### 1. 乙类互补功放电路
![[Excalidraw/第五章 基本放大电路 2023-11-28 09.26.20|1200]]
上图分别为乙类和OCL互补对称功率放大电路, 在电流的负半周时, 通过外加的三极管Q5实现负半周时的功率放大输出。通过双电源使得两个三级管处于相应偏状态。

但是由于每个三级管都有死区电压, 因此输出电压会发生**交越失真**, 即在正偏电压小于死区电压时发生失真。
#### 2. OCL甲乙类互补功放电路
OCL互补功率放大电路表示output-capacitor-less complementary symmetry power amplifier, 即输出端不接电容。
为了避免乙类功率放大电路的交越失真现象, **通过两个二极管使得$T_2, T_3$均处于正偏状态(通过二极管钳位作用使得预先有电压降)使得$T_2, T_{3}$导通, $T_1$处在甲类工作状态**, 从而**分别对$T_1$先放大的的正向和反向输出电压再次分别放大** 

工作时集射极之间的饱和电压为$U_{CES}$, 为了不出现饱和失真， 则最大输出电压为$U_{m}= U_{CC} - U_{CES}$
$$p_{o\max} = \frac{(U_{CC} - U_{CES })^{2}}{2 R_{L}}$$
OCL功放电路的缺点是**需要双电源供电**。
