如果将变量和某个确定的值做比较，可以使用如下方法:
```shell 
if [ "${DEMO_RUN}"x == "1"x ]
# 这里的 `x` 是为了确保比较的字符串不会因为空值而导致错误。在 Bash 脚本中，`x` 是一个惯用的技巧，用于将变量的值与字符串进行比较。具体来说: 
# `${DEMO_RUN}` 表示变量 `DEMO_RUN` 的值。
#`"${DEMO_RUN}"x` 将变量值转换为字符串，并在其后添加一个字符 `x`。这样，即使 `DEMO_RUN` 是空值，也不会导致比较错误。 
```
