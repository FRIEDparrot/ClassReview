## 一、lambda表达式的基本操作
5.4 以前的部分需要在pro中加入
```cpp 
CONFIG += c++11
```

在lambda 表达式中 ，如果使用 mutable 修饰符， 用于**修饰值传递的变量**，**修改的是原值的拷贝**，但**不修改变量本身**
另外需要注意的是，在函数后面加上()， 表示调用函数。
```cpp 
[](parameters] mutable -> returnValue(
)
// mutable修改值传递变量，修改拷贝， 不修改本体

// 注意btn2 是局部变量， 如果需要进行外部传入参数, 则在[]中加入 = , 也可以使用 & 引用传递(), 推荐值传递
[=](){  // 使用值传递的方法
btn2->setText("Push Button to Close");
}();   // 声明函数, 后面加上()表示调用函数
// 除了= 也可以使用this

// 完整写法如下: 
[=]()mutable -> void{  // 使用值传递的方法
	btn2->setText("Push Button to Close");
}(); 
```

其中 `[]` 内的表达式可以如下: 

| `[]`中的表达式 | 含义                               |
| -------------- | ---------------------------------- |
| =              | 值传递局部变量                     |
| &              | 引用传递局部变量                   |
| this           | 函数体内可以使用Lambda成员变量     |
| a  , &b        | 将a按照值进传递, b按照引用进行传递 |
| =, &a, &b      | 将a,b按照引用传递, 其余按照值传递  | 

比如可以使用如图的方法在按钮里面之间调用一个lambda函数，只需要将函数名传入即可(一般则类::函数名), 常见的方法有直接放在Button里面, 以及使用返回值
```cpp 
QPushButton *btn3 = new QPushButton(this);
btn3->move(20,300);
connect(btn3, &QPushButton::clicked, this, [=](){   // 注意使用 = 进行值传递
	btn3->setText("You Pushed this Button");
	btn3->resize(300, 30);
});

// 返回值
int test = []()->int{return 100;}(); // 注意需要进行调用函数，而不是只声明
```

需要注意的是, 值传递和引用传递, 默认是只读，可以修改值传递部分的拷贝，但不修改关键字本身

在传递时， 使用& 可能会出错

通过继承和lambda表达式消息等操作，可以实现窗口的新建等等
