### 🗒️ Templates（模板文件夹）
**本文件夹包含一些实用的快捷键模板，用于配合 Templater 插件使用以提高笔记效率**

使用方法：
- 按 `Ctrl+P` 打开命令面板，可以搜索所有可用的模板以及相关的符号
- 通过自定义快捷键快速插入常用符号和格式  

#### 常用快捷键示例：
**标题快捷键：**
- `Alt+Shift+1` ~ `Alt+Shift+6`：插入一级到六级标题

**数学符号：**
- `Alt+A`：插入 α（alpha）
- `Alt+B`：插入 β（beta）
- `Alt+D`：插入 δ（delta）
- `Alt+Shift+D`：插入 Δ（Delta大写）
- `Alt+T`：插入 τ（tau）
- `Alt+Shift+T`：插入 θ（theta）
- `Alt+L`：插入 λ（lambda）
- `Alt+Shift+L`：插入 Λ（Lambda大写）
- `Alt+S`：插入 σ（sigma）
- `Alt+Shift+S`：插入 Σ（Sigma大写）
- `Alt+W`：插入 ω（omega）
- `Alt+Shift+W`：插入 Ω（Omega大写）

**数学运算：**
- `Alt+Shift+F`：插入分数（fraction）
- `Alt+I`：插入积分符号
- `Alt+Shift+I`：插入带上下限的积分
- `Ctrl+Shift+S`：插入求和符号（无索引）
- `Alt+Ctrl+Shift+S`：插入求和符号（带索引）
- `Alt+Shift+P`：插入连乘符号
- `Alt+Ctrl+P`：插入偏导数
- `Ctrl+Shift+P`：插入二阶导数
- `Alt+Shift+N`：插入梯度算子 ∇（nabla）
- `Ctrl+Shift+L`：插入极限符号

**括号与括起：**
- `Alt+9`：插入圆括号
- `Alt+[`：插入方括号
- `Ctrl+Shift+[`：插入大括号
- `Alt+Shift+[`：插入基本大括号
- `Alt+\`：插入竖线括号
- `Alt+Shift+\`：插入绝对值括号

**三角函数：**
- `Alt+Shift+X`：插入 sin
- `Alt+Shift+C`：插入 cos
- `Alt+Shift+V`：插入 tan
- `Ctrl+Shift+C`：插入 csc（余割）
- `Ctrl+Shift+X`：插入 sec（正割）
- `Ctrl+Shift+V`：插入 cot（余切）

**集合符号：**
- `Alt+Ctrl+,`：插入属于符号 ∈
- `Ctrl+Shift+,`：插入不属于符号 ∉
- `Alt+Ctrl+U`：插入并集符号 ∪
- `Ctrl+Shift+U`：插入交集符号 ∩
- `Alt+Ctrl+9`：插入真子集符号
- `Alt+Ctrl+0`：插入真超集符号

**比较符号：**
- `Alt+,`：插入小于等于 ≤
- `Alt+.`：插入大于等于 ≥
- `Alt+=`：插入恒等符号 ≡
- `Alt+Shift+=`：插入不恒等符号

**其他实用功能：**
- `Ctrl+Shift+E`：插入数学公式块
- `Ctrl+Shift+Q`：插入带框数学公式
- `Ctrl+Shift+W`：插入分段函数
- `Alt+H`：插入橙色高亮文本
- `Alt+Ctrl+H`：插入蓝色高亮文本
- `Ctrl+Shift+R`：插入提示框（Admonition）
- `Alt+Shift+Enter`：插入代码块
- `Alt+Shift+R`：插入表格

**界面控制：**
- `Ctrl+[`：切换左侧边栏
- `Ctrl+]`：切换右侧边栏
- `Ctrl+9`：切换应用功能区
- `Ctrl+0`：切换右下统计隐藏状态
- `F2`：切换预览模式
- `F3`：切换源码模式
- `F6`：切换暗色/亮色主题
- `Ctrl+K`：打开全局搜索\notin

## 🚀 使用建议

1. **使用 Obsidian 打开本仓库**：本仓库专为 Obsidian 优化，建议使用 Obsidian 打开以获得最佳体验
2. **探索快捷键**：按 `Ctrl+P` 打开命令面板，熟悉各种模板快捷键，可大幅提升笔记效率
3. **查看热键配置**：所有快捷键配置位于 `.obsidian/hotkeys.json` 文件中
4. **双向链接**：充分利用 Obsidian 的双向链接功能，构建知识网络
5. **搜索功能**：使用 `Ctrl+K` 快速搜索笔记内容
